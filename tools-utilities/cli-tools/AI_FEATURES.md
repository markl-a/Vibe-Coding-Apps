# 🤖 AI 增強功能說明

本文檔詳細說明 CLI Tools 中各工具的 AI 輔助功能。

## 概述

所有工具都使用 AI 輔助開發，部分工具還整合了智能分析和建議功能，提供更好的使用者體驗。

## 工具 AI 功能詳解

### 1. 🖥️ sysmon - 系統監控工具

#### AI 健康分析
- **健康評分系統（0-100 分）**
  - 綜合評估 CPU、記憶體、磁碟使用狀況
  - 根據系統負載自動扣分
  - 提供直觀的健康狀態評級

- **智能問題檢測**
  - 自動檢測 CPU 使用率異常（>80%、>90%）
  - 記憶體使用警告（>85%、>90%）
  - 磁碟空間不足提醒（>90%、>95%）
  - Swap 使用率監控

- **智能建議**
  - 根據系統狀態提供具體建議
  - 推薦優化措施和解決方案
  - 預防性維護建議

#### 使用範例
```bash
# 執行 AI 健康分析
python sysmon.py --analyze

# 輸出範例：
# 健康評分: 85/100 (良好)
# 發現的問題:
#   🟡 記憶體使用率偏高 (>85%)
# 建議:
#   建議：關閉不必要的應用程式，清理系統快取
```

### 2. 🔧 githelper - Git 輔助工具

#### AI 提交訊息建議
- **智能分析變更**
  - 分析修改、新增、刪除的檔案數量
  - 識別檔案類型（Python、JavaScript、文檔等）
  - 參考歷史提交模式

- **自動生成建議**
  - 根據變更類型建議提交前綴（feat/fix/docs/chore）
  - 考慮檔案類型生成具體訊息
  - 遵循團隊的提交習慣

#### 提交模式分析
- **統計提交類型分布**
  - feat、fix、docs、style、refactor、test、chore
  - 視覺化圖表顯示
  - 百分比統計

- **作者貢獻分析**
  - 統計各作者的提交次數
  - 計算貢獻百分比
  - 視覺化圖表

- **活躍度分析**
  - 計算活躍天數
  - 平均每天提交數
  - 開發頻率趨勢

#### 分支清理建議
- 自動識別已合併分支
- 排除主分支（main/master/develop）
- 提供安全的清理建議

#### 使用範例
```bash
# AI 建議提交訊息
python githelper.py suggest

# 輸出範例：
# 1. feat: Follow recent commit pattern
# 2. feat(python): Update Python code
# 3. refactor: Update existing files

# 顯示統計資訊（含 AI 分析）
python githelper.py stats
```

### 3. 🔍 jsonql - JSON 查詢工具

#### AI 查詢建議
- **智能分析 JSON 結構**
  - 自動識別資料類型（物件/陣列）
  - 列舉主要鍵名
  - 識別嵌套結構

- **生成建議查詢**
  - 基本查詢（整個文檔）
  - 常用欄位查詢
  - 陣列元素查詢
  - 過濾查詢範例

#### AI 結構分析
- **深度分析**
  - 計算 JSON 深度
  - 統計總元素數
  - 計算嵌套物件和陣列數量
  - 分析資料大小

- **提供洞察**
  - 資料類型資訊
  - 鍵數量和範例
  - 陣列長度
  - 優化建議

#### 智能統計
- **數值欄位統計**
  - 自動識別數值欄位
  - 遞迴搜尋所有層級
  - 計算完整統計資訊：
    - 數量、總和、平均值
    - 最小值、最大值
    - 中位數、範圍

#### 使用範例
```bash
# AI 建議查詢
python jsonql.py data.json --suggest

# 輸出範例：
# 1. $ - 獲取整個文檔
# 2. $.users - 獲取 'users' 的值
# 3. $.users[*] - 獲取 'users' 列表中的所有元素
# 4. $.users[*].name - 獲取所有 'name' 值

# AI 結構分析
python jsonql.py data.json --analyze

# 統計分析
python jsonql.py data.json --stats age
```

### 4. 🔐 passgen - 密碼生成器

#### 密碼強度評估
- **多維度評估**
  - 長度評分
  - 字符複雜度評分
  - 熵值計算

- **安全性分析**
  - 檢測小寫字母
  - 檢測大寫字母
  - 檢測數字
  - 檢測特殊符號

- **評級系統**
  - 非常強（8-10 分）
  - 強（6-7 分）
  - 中等（4-5 分）
  - 弱（2-3 分）
  - 非常弱（0-1 分）

#### 使用範例
```bash
# 生成密碼並顯示強度
python passgen.py --show-strength

# 檢查現有密碼
python passgen.py --check "MyPassword123"
```

## 技術實現

### AI 演算法說明

1. **健康評分算法（sysmon）**
   ```python
   健康分數 = 100
   - CPU > 90% 扣 30 分
   - CPU > 80% 扣 15 分
   - 記憶體 > 90% 扣 30 分
   - 記憶體 > 85% 扣 15 分
   - 磁碟 > 95% 扣 25 分
   - 磁碟 > 90% 扣 10 分
   ```

2. **提交訊息建議（githelper）**
   ```
   1. 分析變更類型（新增/修改/刪除）
   2. 識別檔案類型
   3. 參考歷史模式
   4. 生成多個建議選項
   ```

3. **查詢建議（jsonql）**
   ```
   1. 解析 JSON 結構
   2. 識別資料類型
   3. 列舉可查詢路徑
   4. 生成 JSONPath 表達式
   ```

## 常見問題

**Q: AI 功能需要網路連線嗎？**
A: 不需要。所有 AI 功能都在本地執行，不需要網路連線。

**Q: AI 建議的準確度如何？**
A: 準確度取決於具體場景，通常在 70-90% 之間。建議作為參考使用。

**Q: 可以自訂 AI 行為嗎？**

**Q: AI 功能會影響性能嗎？**
A: 影響極小，所有演算法都經過優化，通常在毫秒級完成。
