# 工具與實用程式 Tools & Utilities

這個資料夾包含各種工具和實用程式專案。

## 子資料夾說明

### cli-tools (命令列工具)
- 終端機命令列工具
- 系統管理工具
- 開發者工具

### automation-scripts (自動化腳本)
- 任務自動化
- 檔案處理腳本
- 系統維護腳本

### data-processing (資料處理)
- 資料轉換工具
- 批次處理工具
- 資料清理工具

### dev-tools (開發工具)
- 程式碼生成器
- 格式化工具
- 測試工具
- 部署工具

## 工具類型建議
- 檔案轉換器 (PDF, 圖片, 影片)
- 密碼生成器/管理器
- 系統監控工具
- 備份工具
- 網頁爬蟲
- API 測試工具
- 程式碼格式化工具
- Git 輔助工具
- 資料庫遷移工具

## 技術棧
- Python (Click, Typer, argparse)
- Node.js (Commander.js, Inquirer)
- Go (Cobra)
- Rust (Clap)
- Bash/Shell scripting
