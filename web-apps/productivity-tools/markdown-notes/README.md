# 📝 Markdown 筆記 | Markdown Notes

一個功能強大、界面美觀的 Markdown 筆記應用，支援即時預覽、標籤管理和程式碼高亮。

## ✨ 功能特色

### 核心功能
- 📝 **Markdown 編輯**
  - 即時預覽
  - 分割視圖
  - 語法高亮
  - 工具列快捷操作

- 📋 **筆記管理**
  - 創建、編輯、刪除筆記
  - 快速搜尋
  - 最愛收藏
  - 標籤分類

- 🎨 **視圖模式**
  - ✏️ 編輯模式
  - ⚡ 分割模式
  - 👁️ 預覽模式

- 🏷️ **標籤系統**
  - 自定義標籤
  - 標籤計數
  - 快速篩選

### Markdown 支援

- **基本語法**
  - 標題（H1-H6）
  - 粗體、斜體、刪除線
  - 引用
  - 連結和圖片

- **進階語法**
  - 程式碼區塊（語法高亮）
  - 表格
  - 任務列表
  - 有序/無序列表

- **GFM 擴展**
  - GitHub Flavored Markdown
  - 自動換行
  - 程式碼語言標註

### 編輯器功能

- 🛠️ **工具列**
  - 快速格式化按鈕
  - 插入連結、圖片
  - 列表和表格
  - 程式碼區塊

- ⌨️ **鍵盤快捷鍵**
  - `Ctrl/Cmd + N` - 新增筆記
  - `Ctrl/Cmd + S` - 儲存筆記
  - `Ctrl/Cmd + F` - 搜尋
  - `Ctrl/Cmd + B` - 粗體
  - `Ctrl/Cmd + I` - 斜體
  - `Ctrl/Cmd + K` - 插入連結

- 📊 **統計資訊**
  - 字元數
  - 字數
  - 行數
  - 最後儲存時間

### 數據管理

- 💾 **本地存儲**
  - 自動儲存
  - 數據持久化

- 📤 **匯入/匯出**
  - JSON 格式匯出
  - 批量匯入
  - 備份還原

## 🚀 快速開始

### 直接使用
1. 用瀏覽器打開 `index.html`
2. 開始創建你的第一個筆記

### 使用本地伺服器
```bash
# Python 3
python -m http.server 8000

# Node.js
npx http-server -p 8000

# PHP
php -S localhost:8000
```

訪問 `http://localhost:8000`

## 📖 使用說明

### 創建筆記

1. 點擊「➕ 新增筆記」按鈕
2. 輸入筆記標題
3. 在編輯器中撰寫 Markdown 內容
4. 自動儲存

### 編輯筆記

**使用工具列**：
- 點擊工具列按鈕快速插入格式
- 選取文字後點擊按鈕應用格式

**使用快捷鍵**：
- `Ctrl/Cmd + B` - 粗體
- `Ctrl/Cmd + I` - 斜體
- `Ctrl/Cmd + K` - 連結

**直接輸入 Markdown**：
```markdown
# 標題 1
## 標題 2

**粗體文字**
*斜體文字*
~~刪除線~~

- 列表項目 1
- 列表項目 2

1. 有序列表 1
2. 有序列表 2

- [ ] 待辦事項 1
- [x] 已完成事項

> 引用文字

`程式碼`

```javascript
// 程式碼區塊
console.log('Hello World');
```

[連結](https://example.com)
![圖片](https://example.com/image.jpg)

| 表頭 1 | 表頭 2 |
| ------ | ------ |
| 內容 1 | 內容 2 |
```

### 管理標籤

1. 在標籤輸入框輸入標籤名稱
2. 按 Enter 鍵添加
3. 點擊標籤旁的 × 移除
4. 點擊側邊欄的標籤快速篩選

### 篩選筆記

**左側邊欄**：
- 📚 所有筆記
- ⭐ 我的最愛
- 🕒 最近編輯（7天內）

**搜尋**：
在搜尋框輸入關鍵字，搜尋標題、內容和標籤

### 視圖切換

- ✏️ **編輯模式** - 只顯示編輯器
- ⚡ **分割模式** - 同時顯示編輯器和預覽（推薦）
- 👁️ **預覽模式** - 只顯示預覽

### 數據管理

**匯出**：
- 點擊「💾 匯出」按鈕
- 下載 JSON 格式備份文件

**匯入**：
- 點擊「📥 匯入」按鈕
- 選擇之前匯出的 JSON 文件

## 📱 Markdown 語法速查

### 標題
```markdown
# H1 標題
## H2 標題
### H3 標題
```

### 文字格式
```markdown
**粗體**
*斜體*
~~刪除線~~
`行內程式碼`
```

### 列表
```markdown
- 無序列表
  - 子項目

1. 有序列表
2. 第二項

- [ ] 待辦事項
- [x] 已完成
```

### 連結和圖片
```markdown
[連結文字](URL)
![圖片替代文字](圖片URL)
```

### 引用
```markdown
> 這是引用文字
```

### 程式碼
````markdown
```javascript
function hello() {
  console.log('Hello World');
}
```
````

### 表格
```markdown
| 欄位 1 | 欄位 2 |
| ------ | ------ |
| 內容 1 | 內容 2 |
```
