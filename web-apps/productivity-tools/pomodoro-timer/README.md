# 🍅 番茄鐘計時器 | Pomodoro Timer

一個簡潔美觀的番茄鐘計時器應用，幫助你提升專注力和工作效率。

## ✨ 功能特色

### 核心功能
- ⏱️ **三種計時模式**
  - 工作模式（預設 25 分鐘）
  - 短休息（預設 5 分鐘）
  - 長休息（預設 15 分鐘）

- 🎯 **智能切換**
  - 完成 4 個番茄鐘後自動進入長休息
  - 支援自動開始休息模式

- 📊 **統計追蹤**
  - 記錄完成的番茄鐘數量
  - 追蹤總專注時間
  - 數據持久化保存

- 🎨 **視覺效果**
  - 圓環進度指示器
  - 不同模式顏色區分
  - 流暢的動畫效果

- 🔔 **通知提醒**
  - 音效通知
  - 瀏覽器桌面通知
  - 可自定義開關

### 進階功能
- ⚙️ **自定義設定**
  - 調整各模式時長
  - 開關音效通知
  - 自動開始休息

- ⌨️ **鍵盤快捷鍵**
  - 空格鍵：開始/暫停
  - R 鍵：重置計時器

- 💾 **本地存儲**
  - 自動保存設定
  - 保存統計數據
  - 無需註冊登入

## 🚀 快速開始

### 直接使用
1. 用瀏覽器打開 `index.html`
2. 開始使用番茄鐘計時器

### 或使用本地伺服器
```bash

python -m http.server 8000

# 使用 Node.js (需安裝 http-server)
npx http-server -p 8000

# 使用 PHP

```

然後訪問 `http://localhost:8000`

## 📖 使用說明

### 基本使用流程

1. **選擇模式**
   - 點擊頂部的「工作」、「短休息」或「長休息」按鈕

2. **開始計時**
   - 點擊「開始」按鈕啟動計時器
   - 可隨時點擊「暫停」暫停計時

3. **重置計時器**
   - 點擊「重置」按鈕重置當前模式的時間

4. **完成番茄鐘**
   - 計時結束會自動播放提示音
   - 顯示桌面通知（需授權）
   - 自動切換到休息模式

### 自定義設定

在右側設定面板中可以調整：
- 工作時長（1-60 分鐘）
- 短休息時長（1-30 分鐘）
- 長休息時長（1-60 分鐘）
- 開關音效通知
- 開關自動開始休息

所有設定會自動保存到瀏覽器本地存儲。

## 🎯 番茄鐘工作法

番茄鐘技術（Pomodoro Technique）是一種時間管理方法：

1. **選擇一個任務**
2. **設定番茄鐘（25 分鐘）**
3. **專注工作直到鈴聲響起**
4. **短暫休息（5 分鐘）**
5. **每 4 個番茄鐘後，休息更長時間（15-30 分鐘）**

### 使用技巧

- 🎯 **專注單一任務** - 一個番茄鐘只做一件事
- 🚫 **避免中斷** - 番茄鐘期間不要被打斷
- 📝 **記錄干擾** - 記下想到的其他事項，稍後處理
- ✅ **完成慶祝** - 每完成一個番茄鐘給自己鼓勵
- 📊 **追蹤進度** - 用統計數據檢視工作效率
