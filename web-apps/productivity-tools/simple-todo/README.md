# ✅ 簡單待辦事項 | Simple Todo

一個功能完整、界面美觀的待辦事項應用，幫助你高效管理日常任務。

## ✨ 功能特色

### 核心功能
- ✅ **任務管理**
  - 快速添加任務
  - 編輯任務詳情
  - 刪除任務
  - 標記完成/未完成

- 📅 **日期管理**
  - 設定截止日期
  - 今天任務檢視
  - 即將到來任務
  - 逾期提醒

- 🎯 **優先級系統**
  - 🔴 緊急
  - 🟠 高優先級
  - 🟡 中優先級
  - 🟢 低優先級

- 🔍 **搜尋與篩選**
  - 全文搜尋任務
  - 按狀態篩選
  - 按優先級篩選
  - 多種排序方式

### 進階功能
- 📊 **統計追蹤**
  - 任務總數
  - 完成數量
  - 各類別計數

- 💾 **數據管理**
  - 本地存儲
  - 數據匯出（JSON）
  - 批量清除已完成

- ⌨️ **鍵盤快捷鍵**
  - `Ctrl/Cmd + N` - 新增任務
  - `Ctrl/Cmd + K` - 聚焦搜尋
  - `Escape` - 關閉模態框

- 🎨 **美觀界面**
  - 現代化設計
  - 流暢動畫
  - 響應式佈局
  - 色彩編碼

## 🚀 快速開始

### 直接使用
1. 用瀏覽器打開 `index.html`
2. 開始管理你的待辦事項

### 使用本地伺服器
```bash
# Python 3
python -m http.server 8000

# Node.js
npx http-server -p 8000

# PHP
php -S localhost:8000
```

訪問 `http://localhost:8000`

## 📖 使用說明

### 添加任務

1. 在頂部輸入框輸入任務標題
2. 可選：設定截止日期
3. 可選：選擇優先級
4. 點擊「添加任務」或按 Enter 鍵

### 管理任務

- **標記完成**: 點擊任務前的勾選框
- **編輯任務**: 點擊 ✏️ 圖標
- **刪除任務**: 點擊 🗑️ 圖標

### 篩選任務

**左側邊欄列表**:
- 📝 所有任務
- 📅 今天
- 📆 即將到來
- ✅ 已完成

**按優先級篩選**:
點擊優先級項目即可篩選

### 搜尋任務

在搜尋框輸入關鍵字，會即時搜尋任務標題和描述。

### 排序任務

支援多種排序方式：
- 創建時間（預設）
- 截止日期
- 優先級
- 標題

### 編輯任務詳情

點擊任務的編輯按鈕可以修改：
- 任務標題
- 任務描述
- 截止日期
- 優先級

### 數據管理

**清除已完成**:
批量刪除所有已完成的任務

**匯出數據**:
將所有任務匯出為 JSON 文件

## 💡 使用技巧

### 高效工作流程

1. **晨間規劃**
   - 查看「今天」任務
   - 設定優先級
   - 估計完成時間

2. **執行任務**
   - 從高優先級開始
   - 一次專注一個任務
   - 及時標記完成

3. **晚間回顧**
   - 檢視完成情況
   - 調整未完成任務
   - 規劃明天

### 任務組織建議

- 使用優先級區分重要性和緊急性
- 大任務分解成小任務
- 設定合理的截止日期
- 定期清理已完成任務

### 最佳實踐

- **保持簡潔**: 任務標題簡短明確
- **及時更新**: 完成後立即標記
- **定期回顧**: 每週回顧調整
- **優先重要**: 先做重要的事
