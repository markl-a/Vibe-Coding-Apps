# ⏱️ 時間追蹤器 - Time Tracker

一個功能強大的時間追蹤工具，幫助你追蹤工作時間、分析生產力模式，並通過 AI 智能分析提供個性化建議。

## ✨ 核心功能

### 1. 時間追蹤
- ⏰ 精確的計時器（時/分/秒）
- ⏸️ 暫停/繼續功能
- 📝 任務描述記錄
- 🏷️ 專案分類（工作、學習、個人、會議、休息）
- 📊 實時顯示今日總時長

### 2. 時間記錄管理
- 📅 自動記錄所有時間條目
- 🗑️ 刪除單個記錄
- 💾 匯出為 JSON 格式
- 🔄 清除全部記錄

### 3. 統計分析
- 📊 今日/本週/本月/總計統計
- 📈 按專案分類統計
- 🕐 時間分布分析

### 4. 🤖 AI 智能分析

#### 最佳工作時段識別
- 分析你的工作習慣
- 識別最高效的時間段
- 提供個性化時間規劃建議

#### 專案效率分析
- 統計各專案投入時間
- 識別最高效的專案類型
- 優化時間分配建議

#### 效率分數計算
- 基於工作時長和頻率
- 實時更新效率評分
- 智能建議提升效率的方法

## 🚀 快速開始

### 線上使用
直接打開 `index.html` 文件即可使用，無需安裝任何依賴。

### 本地開發
```bash
# 克隆項目
git clone [repo-url]

# 進入目錄
cd time-tracker

# 使用任意 HTTP 服務器運行
python -m http.server 8000
# 或
npx serve
```

訪問 `http://localhost:8000`

## 📖 使用指南

### 開始追蹤時間

1. **輸入任務名稱**
   - 在「你在做什麼？」輸入框中輸入當前任務

2. **選擇專案分類**
   - 從下拉選單選擇專案類型（可選）

3. **開始計時**
   - 點擊「開始」按鈕開始追蹤時間

4. **暫停/繼續**
   - 需要暫停時點擊「暫停」按鈕
   - 繼續工作時再次點擊「繼續」

5. **停止並記錄**
   - 完成任務後點擊「停止」按鈕
   - 時間記錄會自動保存

### 查看統計數據

- **統計卡片**：顯示今日、本週、本月和總計時長
- **時間記錄列表**：查看所有歷史記錄
- **AI 智能分析**：獲取個性化的生產力洞察

### 匯出數據

1. 點擊「匯出」按鈕
2. 下載 JSON 格式的數據文件
3. 數據包含所有時間記錄和統計信息

## 🎯 AI 智能功能說明

### 最佳工作時段 AI
- **數據收集**：自動記錄每次工作的時間段
- **模式識別**：分析你在不同時間段的工作時長
- **智能推薦**：識別你最高效的 2 個時間段

### 專案效率 AI
- **專案統計**：計算每個專案的總投入時間
- **效率排名**：識別你最常使用的專案類型
- **優化建議**：根據數據提供時間分配建議

### 效率分數算法
```
效率分數 = min(100, (平均工作時長(小時) × 20))

評級標準：
- 80-100分：效率極高 🎉
- 60-79分：效率良好 👍
- 40-59分：有提升空間 💪
- 0-39分：需要改進 🎯
```

### AI 建議邏輯

根據你的使用數據，AI 會提供以下類型的建議：

1. **高效率用戶（分數 ≥ 80）**
   - 鼓勵繼續保持當前節奏

2. **中等效率用戶（分數 60-79）**
   - 建議在高效時段安排重要任務

3. **短時工作用戶（平均 < 1小時）**
   - 建議延長專注時間

4. **需要改進用戶**
   - 推薦番茄鐘技術等時間管理方法
