# 網頁應用 Web Applications

這個資料夾包含各種網頁應用專案。

## 子資料夾說明

### portfolio-blog (作品集/部落格)
- 個人作品集網站
- 部落格系統
- 個人品牌網站

### e-commerce (電子商務)
- 購物網站
- 商品展示平台
- 線上商店系統

### social-media (社交媒體)
- 社交網絡應用
- 論壇系統
- 即時聊天應用

### dashboard-analytics (儀表板/分析)
- 數據視覺化儀表板
- 管理後台
- 分析工具

### productivity-tools (生產力工具)
- 待辦事項應用
- 筆記系統
- 專案管理工具
- 時間追蹤器

### landing-pages (著陸頁)
- 產品著陸頁
- 活動宣傳頁面
- SaaS 產品頁面

## 技術棧建議
- Frontend: React, Vue, Next.js, Svelte, Angular
- Styling: Tailwind CSS, Material-UI, Styled Components
- Backend: Node.js, Python (Django/Flask), Ruby on Rails
- Database: PostgreSQL, MongoDB, Firebase
