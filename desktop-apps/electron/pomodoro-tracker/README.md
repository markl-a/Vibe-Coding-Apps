# 🍅 Pomodoro Tracker - 專注時間管理工具

> 🤖 **AI-Driven | AI-Native** 🚀

基於番茄工作法的時間管理桌面應用,幫助你提升專注力和工作效率。

## ✨ 功能特色

### 核心功能
- ⏱️ **番茄鐘計時器**: 標準 25 分鐘工作 + 5 分鐘休息
- 🔔 **通知提醒**: 工作和休息時段結束時的系統通知
- 📊 **統計追蹤**: 每日、每週、每月完成的番茄鐘數量
- 📝 **任務管理**: 為每個番茄鐘設定待辦任務
- 🎯 **目標設定**: 設定每日番茄鐘目標

### 進階功能
- 📈 **數據視覺化**: 圖表顯示生產力趨勢
- 🎨 **自訂時間**: 自訂工作和休息時長
- 🔊 **音效提示**: 可自訂的提示音效
- 🌙 **深色模式**: 護眼深色主題
- 💾 **資料匯出**: 匯出統計數據為 CSV/JSON
- 🏆 **成就系統**: 達成目標解鎖成就徽章
- ⏸️ **暫停功能**: 緊急情況可暫停計時

### 番茄工作法特色
- ✅ 長休息時段 (每 4 個番茄鐘)
- ✅ 自動循環計時
- ✅ 工作階段歷史記錄
- ✅ 中斷記錄和分析
- ✅ 專注模式 (隱藏干擾)

## ⏱️ 番茄工作法介紹

番茄工作法是一種時間管理技巧,由 Francesco Cirillo 在 1980 年代末期發明。

### 工作流程

1. **選擇任務**: 確定要完成的任務
2. **設定番茄鐘**: 設定 25 分鐘計時器
3. **專注工作**: 直到計時器響起
4. **休息**: 休息 5 分鐘
5. **重複**: 每完成 4 個番茄鐘,長休息 15-30 分鐘

### 核心原則

- 🎯 一次只專注一個任務
- ⏸️ 避免中斷和分心
- 📊 追蹤和記錄進度
- 🔄 定期休息保持效率

## ⚙️ 設定選項

應用會在使用者資料夾建立設定檔:
- Windows: `%APPDATA%/pomodoro-tracker/config.json`
- macOS: `~/Library/Application Support/pomodoro-tracker/config.json`
- Linux: `~/.config/pomodoro-tracker/config.json`

可配置項目:
```json
{
  "workDuration": 1500,
  "shortBreak": 300,
  "longBreak": 900,
  "pomodorosUntilLongBreak": 4,
  "autoStartBreaks": true,
  "autoStartPomodoros": false,
  "soundEnabled": true,
  "notificationsEnabled": true,
  "tickSound": false,
  "theme": "light",
  "dailyGoal": 8
}
```

### 時間設定說明

- **workDuration** (1500秒 = 25分鐘): 工作時段長度
- **shortBreak** (300秒 = 5分鐘): 短休息時段
- **longBreak** (900秒 = 15分鐘): 長休息時段
- **pomodorosUntilLongBreak** (4): 幾個番茄鐘後長休息

## 💡 使用技巧

### 1. 設定合理的每日目標

- 初學者: 4-6 個番茄鐘/天
- 進階者: 8-10 個番茄鐘/天
- 專家: 12+ 個番茄鐘/天

### 2. 選擇合適的任務

- 選擇需要專注的任務
- 將大任務拆分成多個番茄鐘
- 避免在番茄鐘中切換任務

### 3. 善用休息時間

- 離開座位走動
- 做簡單的伸展運動
- 喝水、吃點心
- 不要查看工作相關內容

### 4. 處理中斷

- 內部中斷: 記錄後繼續
- 外部中斷: 禮貌地推遲
- 緊急中斷: 暫停番茄鐘

## 🔧 快捷鍵

- `Ctrl/Cmd + Q`: 結束應用
