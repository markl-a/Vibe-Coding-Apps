# 桌面應用 Desktop Applications

這個資料夾包含各種桌面應用程式專案。

## 子資料夾說明

### electron
- 跨平台桌面應用 (Windows, macOS, Linux)
- 使用 Web 技術 (HTML, CSS, JavaScript)
- 整合 Node.js API

### tauri
- 輕量級跨平台桌面應用
- 使用 Rust + Web 前端
- 更小的應用體積

### native
- 原生桌面應用
- Windows: C#/.NET, C++
- macOS: Swift/SwiftUI
- Linux: GTK, Qt

## 適合開發的應用類型
- 文字編輯器/IDE
- 音樂/影片播放器
- 筆記應用
- 系統工具
- 檔案管理器
- 設計工具
- 通訊軟體
- 資料庫管理工具
- 螢幕錄製/截圖工具

## 技術棧
- Electron + React/Vue
- Tauri + Svelte/React
- .NET MAUI
- Qt/C++
