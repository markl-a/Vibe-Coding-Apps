# 🍅 Pomodoro 專注計時器

> 🤖 **AI-Driven | AI-Native** 🚀

基於番茄工作法的智能專注計時器，使用 Tauri + React 構建，內建 AI 智能休息建議。

## ✨ 功能特點

### 核心功能
- ⏱️ **番茄工作法計時器**: 25分鐘工作、5分鐘短休息、15分鐘長休息
- 📊 **統計追蹤**: 追蹤每日完成的番茄鐘數和專注時間
- 🔔 **系統通知**: 計時結束時自動通知
- 🎯 **任務管理**: 為每個番茄鐘設定任務目標
- 📈 **數據視覺化**: 查看每日、每週專注統計

### AI 智能功能
- 🤖 **智能休息建議**: AI 根據工作強度建議休息時長
- 💡 **專注力分析**: 分析最佳工作時段
- 🎯 **任務優先級建議**: AI 幫助安排任務順序
- 📝 **工作模式推薦**: 根據歷史數據推薦最佳工作模式

### 進階功能
- 🌙 **深色模式**: 支援淺色/深色主題
- 🔧 **自訂設定**: 自訂工作和休息時長
- 📱 **系統托盤**: 最小化到系統托盤持續計時
- ⌨️ **全局快捷鍵**: 快速控制計時器
- 🎵 **提示音**: 計時結束提示音（可關閉）

## 📖 使用說明

### 番茄工作法基礎

1. **選擇任務**: 選擇一個要完成的任務
2. **開始工作**: 點擊開始，專注工作 25 分鐘
3. **短休息**: 工作結束後休息 5 分鐘
4. **重複循環**: 完成 4 個番茄鐘後，進行 15 分鐘長休息

### AI 智能輔助

#### 智能休息建議
AI 會根據你的工作強度和時段，建議最適合的休息時長：
- 高強度工作後建議較長休息
- 下午時段建議額外休息
- 連續工作後建議活動休息

#### 專注力分析
查看你的專注力熱力圖：
- 一天中哪個時段最專注
- 一週中哪天效率最高
- 最佳工作時長建議

#### 任務優先級
AI 根據以下因素建議任務順序：
- 緊急程度
- 預估完成時間
- 歷史完成率
- 當前時段的專注力

## 📊 數據結構

### 番茄鐘記錄
```rust
struct PomodoroSession {
    id: i64,
    task: String,
    duration: i32,        // 分鐘
    session_type: String, // work, short_break, long_break
    completed: bool,
    started_at: i64,
    finished_at: Option<i64>,
}
```

## ⌨️ 快捷鍵

- `Ctrl/Cmd + S`: 開始/暫停計時器
- `Ctrl/Cmd + R`: 重置計時器
- `Ctrl/Cmd + N`: 新增任務
- `Ctrl/Cmd + ,`: 打開設定

## ⚙️ 設定選項

### 時間設定
- 工作時長（預設 25 分鐘）
- 短休息時長（預設 5 分鐘）
- 長休息時長（預設 15 分鐘）
- 長休息間隔（預設 4 個番茄鐘）

### 通知設定
- 啟用系統通知
- 提示音開關
- 提示音音量

### AI 設定
- 啟用智能休息建議
- 啟用專注力分析
- 啟用任務優先級建議

## 🤖 AI 功能詳解

### 智能休息建議算法
```
考慮因素：
1. 連續工作時長
2. 當前時段（早/午/晚）
3. 工作強度（中斷次數）
4. 歷史模式

建議休息 = 基礎休息時間 + 強度調整 + 時段調整
```

### 專注力分析
AI 分析你的工作模式：
- 統計每小時完成的番茄鐘數
- 計算平均中斷次數
- 識別高效時段
- 生成個性化建議

### 任務優先級
基於多個維度評分：
- 截止時間接近度（40%）
- 預估難度（30%）
- 當前精力狀態（20%）
- 任務依賴關係（10%）
