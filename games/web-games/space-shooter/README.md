# 🚀 太空射擊 (Space Shooter)

使用 Phaser 3 開發的經典太空射擊遊戲,具有波次敵人、道具系統、爆炸特效和漸進難度。

## ✨ 功能特點

### 核心玩法
- 🎮 **經典射擊** - 控制飛船,射擊消滅敵人
- 👾 **波次系統** - 敵人分波出現,難度逐漸增加
- 💣 **爆炸效果** - 擊毀敵人時的視覺反饋
- ❤️ **生命系統** - 3 條生命,被擊中會減少
- 🏆 **計分系統** - 擊毀敵人獲得分數

### 遊戲特性
- ⭐ **道具系統** - 隨機掉落生命補給道具
- 🔫 **敵人 AI** - 敵人會主動射擊攻擊玩家
- 🌟 **星空背景** - 閃爍的星星營造太空氛圍
- 📊 **實時統計** - 顯示分數、生命、關卡等資訊
- 💾 **最高分記錄** - 本地保存你的最佳成績
- 🎯 **無敵時間** - 被擊中後短暫無敵保護

### 技術特色
- 使用 Phaser 3 遊戲引擎
- Arcade Physics 物理系統
- 物件池管理(子彈、敵人)
- 程序化生成圖形
- 粒子效果和動畫
- 碰撞檢測系統

## 🚀 快速開始

### 前置要求

無需安裝任何依賴!遊戲通過 CDN 加載 Phaser.js。

### 運行方式

**方法一:直接打開**
```bash
# 在瀏覽器中打開
open index.html  # macOS
xdg-open index.html  # Linux
start index.html  # Windows
```

**方法二:使用本地伺服器 (推薦)**
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000

# Node.js (http-server)
npx http-server -p 8000

# PHP
php -S localhost:8000
```

然後訪問: `http://localhost:8000`

## 🎮 遊戲操作

### 鍵盤控制
- `← →` **左右移動** - 方向鍵控制飛船移動
- `A D` **左右移動** - WASD 控制(替代方案)
- `空白鍵` **射擊** - 發射子彈攻擊敵人
- `Enter` **開始/重新開始** - 開始遊戲或重新開始

### 遊戲流程
1. **開始** - 按 Enter 開始遊戲
2. **移動** - 左右移動閃避敵人和子彈
3. **射擊** - 持續按空白鍵消滅敵人
4. **收集** - 撿取掉落的道具恢復生命
5. **生存** - 盡可能生存更久,獲得高分

## 📋 遊戲規則

### 得分系統
- 擊毀敵人: **+10 分**
- 收集道具: **+50 分**

### 生命系統
- 初始生命: **3 條**
- 被敵人撞擊: **-1 生命**
- 被敵人子彈擊中: **-1 生命**
- 收集道具: **+1 生命** (最多 5 條)
- 被擊中後有 **2 秒無敵時間**

### 失敗條件
- 生命值歸零

### 難度提升
- 每消滅一波敵人,敵人速度會增加
- 關卡數會持續提升
- 敵人射擊頻率保持穩定

## 🎯 遊戲策略

### 初學者技巧
- 保持在螢幕下方中央位置
- 優先閃避敵人子彈
- 持續射擊,不要停止
- 及時收集生命道具
- 利用無敵時間調整位置

### 進階技巧
- 預判敵人移動軌跡
- 使用左右移動閃避彈幕
- 優先擊毀最接近的敵人
- 在安全時才收集道具
- 觀察敵人射擊模式

### 高分秘訣
- 不漏掉任何敵人
- 保持連續擊殺
- 合理使用無敵時間
- 最大化道具獎勵分數
- 盡早消滅敵人避免彈幕累積

### 遊戲架構

```
場景生命週期:
preload() → create() → update()

遊戲循環:
update() 每幀執行
├── 玩家移動控制
├── 子彈管理
├── 敵人行為更新
├── 碰撞檢測
└── 清理離屏物件
```

### 物件池系統
```javascript
// 子彈物件池
bullets = this.physics.add.group({
    defaultKey: 'bullet',
    maxSize: 30  // 最多 30 發子彈
});

// 重用物件
const bullet = bullets.get(x, y, 'bullet');
```

### 碰撞檢測
```javascript
// Phaser 重疊檢測
this.physics.add.overlap(bullets, enemies, hitEnemy);
this.physics.add.overlap(player, enemies, hitPlayer);
this.physics.add.overlap(player, enemyBullets, hitPlayerWithBullet);
this.physics.add.overlap(player, powerUps, collectPowerUp);
```

### 敵人生成算法
```javascript
function createEnemyWave() {
    const rows = 3;
    const cols = 6;

```

## 🐛 疑難排解

### 常見問題

**Q: 遊戲運行卡頓?**
A:
- 減少同時存在的子彈數量
- 減少敵人數量
- 關閉 debug 模式

**Q: 射擊速度太快/太慢?**
A: 調整 `shootDelay` 參數

**Q: 敵人太強/太弱?**
A: 調整敵人速度、射擊頻率和數量

**Q: 道具掉落太少?**
A: 增加 `Math.random() < 0.1` 中的機率值
