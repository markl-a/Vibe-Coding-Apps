# 遊戲 Games

這個資料夾包含各種遊戲專案。

## 子資料夾說明

### web-games
- 瀏覽器遊戲
- HTML5 Canvas 遊戲
- WebGL 3D 遊戲

### mobile-games
- iOS/Android 遊戲
- 休閒遊戲
- 益智遊戲

### desktop-games
- PC 遊戲
- 獨立遊戲
- 模擬遊戲

### game-engines
- 使用遊戲引擎開發的專案
- Unity
- Godot
- Unreal Engine

## 遊戲類型建議
- 益智遊戲 (Puzzle Games)
- 平台跳躍遊戲 (Platformers)
- 射擊遊戲 (Shooters)
- 角色扮演遊戲 (RPG)
- 策略遊戲 (Strategy)
- 卡牌遊戲 (Card Games)
- 文字冒險遊戲 (Text Adventures)
- 休閒遊戲 (Casual Games)

## 技術棧
- Phaser.js (Web)
- Three.js/Babylon.js (3D Web)
- Unity + C#
- Godot + GDScript
- Pygame (Python)
