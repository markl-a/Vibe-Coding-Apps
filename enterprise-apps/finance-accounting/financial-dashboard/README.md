# 財務儀表板 (Financial Dashboard)

綜合財務數據分析和視覺化系統，提供企業財務狀況的即時洞察。

## 功能特色

### 📊 數據視覺化
- **即時儀表板**：關鍵財務指標一目了然
- **互動圖表**：使用 Plotly 的互動式圖表
- **趨勢分析**：收入、支出、利潤趨勢
- **多維度分析**：按分類、時間、項目分析

### 💰 財務報表
- **損益表**：收入、成本、利潤分析
- **資產負債表**：資產、負債、權益狀況
- **現金流量表**：三大現金流活動分析
- **財務比率**：流動性、獲利能力、槓桿比率

### 📈 KPI 指標
- 總收入 & 收入增長率
- 總支出 & 成本控制
- 淨利潤 & 利潤率
- 現金流 & 營運資金

## 安裝

```bash
pip install -r requirements.txt
```

## 使用方法

```bash
streamlit run app.py
```

## 核心功能

### 1. 財務總覽
- 總收入、總支出、淨利潤、現金流
- 同比增長率
- 關鍵指標儀表板

### 2. 收入支出分析
- 收入vs支出趨勢圖
- 收入來源分布
- 支出分類分析

### 3. 利潤分析
- 毛利潤 vs 淨利潤
- 利潤率計算
- 營業費用率

### 4. 現金流分析
- 營運、投資、融資活動現金流
- 現金流趨勢
- 現金流預測

### 5. 財務比率
- **流動性比率**：流動比率、速動比率
- **獲利能力**：ROE、ROA、利潤率
- **槓桿比率**：負債比率、負債權益比

## 使用案例

### 中小企業
```
- 每日監控財務狀況
- 每週檢視收支趨勢
- 每月生成財務報表
- 即時掌握現金流
```

### 財務分析師
```
- 快速生成財務報告
- 多維度數據分析
- 趨勢預測和建議
- 導出數據進行深入分析
```

### 投資者
```
- 評估企業財務健康度
- 分析獲利能力
- 審查現金流狀況
- 比較財務比率
```

## 技術架構

- **前端**：Streamlit
- **圖表**：Plotly（互動式圖表）
- **數據處理**：Pandas
- **資料儲存**：JSON
