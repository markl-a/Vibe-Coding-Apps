# 預算規劃器 (Budget Planner) 🎯

智能預算規劃和財務目標管理系統，幫助個人和企業制定、追蹤和優化預算。

## 功能特色

### 📊 預算管理
- **多維度預算設定**：支援月度、季度、年度預算
- **分類預算**：依費用分類設定個別預算限額
- **預算範本**：預設常用預算範本（50/30/20法則等）
- **彈性調整**：根據實際情況動態調整預算

### 🎯 財務目標
- **目標設定**：儲蓄目標、債務清償目標、投資目標
- **進度追蹤**：即時追蹤目標達成進度
- **里程碑設定**：分階段達成大型財務目標
- **目標提醒**：重要里程碑自動提醒

### 📈 預算分析
- **執行率分析**：預算執行情況百分比
- **超支預警**：即時警示超支項目
- **趨勢分析**：歷史預算執行趨勢
- **對比分析**：實際支出 vs 預算對比

### 🤖 AI 智能功能
- **智能預算建議**：基於歷史數據的預算建議
- **異常檢測**：自動識別異常支出
- **預測分析**：預測未來預算需求
- **優化建議**：提供預算優化方案

## 核心功能模組

```python
budget = {

    "total_budget": 60000,

        "儲蓄": 16000
    },
    "start_date": "2025-01-01",
    "end_date": "2025-01-31"
}
```

### 2. 預算追蹤

追蹤實際支出與預算的對比：
- 即時計算剩餘預算
- 計算執行百分比
- 超支警示
- 節省獎勵

### 3. 財務目標管理

```python
goal = {
    "name": "購車基金",
    "type": "savings",  # savings, debt, investment
    "target_amount": 500000,
    "current_amount": 150000,
    "deadline": "2026-12-31",
    "monthly_contribution": 10000,
    "priority": "high"

```

### 4. 預算範本

#### 50/30/20 法則
- 50% 必需品（房租、食物、交通）
- 30% 想要的（娛樂、購物）
- 20% 儲蓄和投資

#### 60/20/20 法則
- 60% 生活開銷
- 20% 儲蓄
- 20% 自由支配

#### 零基預算法
每一塊錢都有明確用途

## 使用流程

### 1. 創建預算

1. 選擇預算期間（月/季/年）
2. 設定總預算金額
3. 分配各分類預算
4. 或選擇預算範本快速設定

### 2. 追蹤預算執行

1. 系統自動整合費用追蹤數據
2. 即時顯示預算執行情況
3. 視覺化圖表展示進度
4. 超支項目自動標記

### 3. 設定財務目標

1. 定義目標類型和金額
2. 設定達成期限
3. 規劃月度貢獻金額
4. 追蹤目標進度

### 4. 分析和優化

1. 查看預算執行報告
2. 分析超支原因
3. 獲取 AI 優化建議
4. 調整下期預算

## AI 智能建議

### 預算優化建議

系統會根據以下因素提供建議：
- 歷史支出模式
- 收入變化趨勢
- 季節性因素
- 財務目標進度

### 自動調整建議

- **收入增加**：建議增加儲蓄比例
- **超支頻繁**：建議重新評估預算分配
- **目標延遲**：建議調整月度貢獻
- **閒置預算**：建議重新分配到優先項目

## 預算健康度評分

系統會根據以下標準評分（滿分 100）：

- **預算執行率** (30分)：在預算範圍內
- **儲蓄率** (25分)：達到收入的 20%
- **債務比率** (20分)：債務低於收入的 30%
- **目標進度** (15分)：財務目標按計劃進行
- **應急基金** (10分)：至少 3-6 個月支出

## 最佳實踐

### 預算設定建議

1. **實際為基礎**：根據過去 3 個月實際支出設定
2. **留有彈性**：預留 10% 緊急預算
3. **定期檢視**：每月檢視並調整
4. **目標導向**：預算應支持財務目標

### 預算執行技巧

1. **即時記錄**：當天記錄支出
2. **週檢查**：每週檢視預算執行
3. **超支處理**：立即分析並調整
4. **慶祝成功**：達成預算目標給予獎勵

## 常見問題

**Q: 如何處理不規律支出？**
A: 計算年度總額後除以 12，每月預留相應金額。

**Q: 超支了怎麼辦？**
A: 分析超支原因，從其他類別調整，或增加收入。

**Q: 多久調整一次預算？**
A: 建議每月檢視，每季度調整一次。

**Q: 如何設定實際的財務目標？**
A: 使用 SMART 原則：具體、可測量、可達成、相關、有時限。
