# 專案儀表板 (Project Dashboard)

一個全面的專案管理儀表板，提供實時專案狀態、KPI 追蹤和數據可視化。

## 功能特點

- 📊 實時專案狀態概覽
- 📈 KPI 指標追蹤
- 🎯 多專案管理和對比
- 👥 團隊績效分析
- ⏱️ 時間和成本追蹤
- 🔔 風險和問題警示
- 📉 趨勢分析和預測
- 💾 自定義報表導出

## 快速開始

```bash
pip install -r requirements.txt
streamlit run src/dashboard.py
```

## 主要功能

### 1. 專案概覽
- 專案進度儀表
- 預算使用情況
- 里程碑追蹤
- 風險等級

### 2. KPI 追蹤
- 專案健康度
- 準時交付率
- 預算符合率
- 團隊生產力

### 3. 資源分析
- 人力資源使用率
- 成本分析
- 工時統計

## 技術棧

- Streamlit / Dash
- Plotly / Chart.js
- Pandas
- SQLite
