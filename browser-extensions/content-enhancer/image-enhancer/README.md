# Image Enhancer Pro 🖼️

強大的圖片增強工具，提供縮放、濾鏡、燈箱查看器、批次下載功能，現在**配備 AI 智能分析**！

## ✨ 功能特色

### 🤖 AI 增強功能 (NEW!)
- **智能圖像分析**：自動檢測圖像類型（風景、肖像、方形、全景）
- **解析度分析**：識別畫質等級和像素數
- **智能標籤**：自動生成描述性標籤（格式、用途、品質）
- **用途建議**：基於分析提供最佳使用建議
- **進度追蹤**：批量下載時的視覺化進度顯示

### 核心功能
- **智能縮放**：滑鼠懸停時放大圖片，支援自訂縮放級別
- **燈箱查看器**：全螢幕圖片瀏覽，支援鍵盤導航和 AI 分析
- **圖片濾鏡**：亮度、對比度、飽和度、模糊等多種濾鏡
- **批次下載**：一鍵下載頁面中所有圖片，帶智能進度顯示
- **圖片搜尋**：快速找出頁面中的大圖

### 進階功能
- **圖片旋轉**：90度增量旋轉圖片
- **裁切工具**：選取圖片區域進行裁切
- **圖片資訊**：顯示圖片尺寸、格式、檔案大小
- **拖曳調整**：直接拖曳改變圖片大小
- **右鍵選單**：快速訪問常用功能
- **快捷鍵支援**：提升操作效率

## 安裝方式

### Chrome/Edge
1. 打開 `chrome://extensions/`
2. 啟用「開發人員模式」
3. 點擊「載入未封裝項目」
4. 選擇此目錄

### Firefox
1. 打開 `about:debugging#/runtime/this-firefox`
2. 點擊「載入暫時附加元件」
3. 選擇 `manifest.json` 文件

## 使用說明

### 基本操作

#### 圖片縮放
- 滑鼠懸停在圖片上自動放大
- 點擊圖片開啟燈箱查看器
- 使用滑鼠滾輪調整縮放級別

#### 燈箱查看器
- 點擊任意圖片開啟
- 使用 `←` `→` 方向鍵切換圖片
- 按 `ESC` 關閉查看器
- 按 `F` 切換全螢幕模式

#### 右鍵選單
- 在圖片上按右鍵顯示選單
- 包含：下載、複製網址、在新分頁開啟等

### 進階操作

#### 套用濾鏡
1. 點擊擴展圖示
2. 選擇「濾鏡」分頁
3. 調整各項參數
4. 點擊「套用到所有圖片」

#### 批次下載
1. 點擊擴展圖示
2. 點擊「批次下載」按鈕
3. 選擇圖片大小閾值（可選）
4. 確認下載

### 鍵盤快捷鍵

在燈箱查看器中：
- `←` / `→` - 上一張/下一張
- `ESC` - 關閉查看器
- `F` - 全螢幕
- `R` - 順時針旋轉
- `Shift + R` - 逆時針旋轉
- `+` / `-` - 放大/縮小
- `0` - 重設縮放
- `D` - 下載當前圖片
- `I` - 顯示圖片資訊
