# Dark Theme Injector

一個智能的深色主題注入器，可自動為任何網站應用深色主題，並支援自訂配色方案。

## 功能特色

- **智能主題注入**：自動分析網站結構並應用深色主題
- **多種預設主題**：提供多種內建深色配色方案
- **自訂配色**：完全可自訂的顏色配置
- **網站白名單/黑名單**：為特定網站啟用或禁用
- **即時切換**：一鍵切換深色/淺色模式
- **保護眼睛**：減少藍光，降低眼睛疲勞
- **圖片處理**：智能調整圖片亮度避免過度曝光
- **效能優化**：使用 CSS 變數和高效選擇器

## 安裝方式

### Chrome/Edge
1. 打開 `chrome://extensions/`
2. 啟用「開發人員模式」
3. 點擊「載入未封裝項目」
4. 選擇此目錄

### Firefox
1. 打開 `about:debugging#/runtime/this-firefox`
2. 點擊「載入暫時附加元件」
3. 選擇 `manifest.json` 文件

## 使用說明

### 基本使用
1. 點擊擴展圖示開啟控制面板
2. 選擇您喜歡的主題
3. 調整亮度、對比度等參數
4. 點擊「應用」按鈕

### 進階設定

#### 網站管理
- 在設定中可以為特定網站建立例外規則
- 支援通配符模式（如 `*.google.com`）

#### 自訂主題
- 點擊「自訂主題」
- 調整背景色、文字色、連結色等
- 儲存為預設主題

### 鍵盤快捷鍵
- `Alt+D` - 切換深色模式
- `Alt+B` - 調整亮度

## 預設主題

1. **經典深色** - 純黑背景，高對比度
2. **午夜藍** - 深藍色調，柔和視覺
3. **暖色調** - 減少藍光，適合夜間使用
4. **高對比** - 最大對比度，提升可讀性
5. **OLED 黑** - 純黑背景，節省 OLED 螢幕電力

## 技術特點

- **CSS 變數系統**：使用 CSS Custom Properties 實現主題切換
- **MutationObserver**：監控 DOM 變化，動態應用樣式
- **智能選擇器**：自動識別並處理各種網站結構
- **效能優化**：使用 requestIdleCallback 避免阻塞主線程
- **儲存同步**：使用 Chrome Storage API 跨裝置同步設定

## 開發

```bash
npm install
npm run build
```

## 已知問題

- 某些使用 Shadow DOM 的網站可能需要特殊處理
- 部分使用內聯樣式的元素可能需要手動調整
