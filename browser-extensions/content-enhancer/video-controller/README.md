# Video Controller Pro

強大的視頻播放控制擴展，提供播放速度調整、截圖、畫中畫和豐富的鍵盤快捷鍵。

## 功能特色

### 播放控制
- **變速播放**：0.25x - 4x 速度調整
- **精確跳轉**：向前/後跳轉 5秒、10秒、30秒
- **音量控制**：快速調整音量
- **畫中畫模式**：支援 PiP 模式
- **循環播放**：AB 段重複播放

### 進階功能
- **視頻截圖**：捕捉當前畫面
- **亮度調整**：調整視頻亮度
- **濾鏡效果**：套用各種視頻濾鏡
- **自動品質**：記住品質偏好設定
- **播放記憶**：記住播放位置

### 鍵盤快捷鍵
- `Space` - 播放/暫停
- `←` / `→` - 後退/前進 5秒
- `Shift + ←/→` - 後退/前進 10秒
- `↑` / `↓` - 增加/減少音量
- `[` / `]` - 減慢/加快播放速度
- `F` - 全螢幕
- `P` - 畫中畫
- `S` - 截圖
- `0-9` - 跳轉到 0%-90%

## 安裝方式

### Chrome/Edge
1. 打開 `chrome://extensions/`
2. 啟用「開發人員模式」
3. 點擊「載入未封裝項目」
4. 選擇此目錄

### Firefox
1. 打開 `about:debugging#/runtime/this-firefox`
2. 點擊「載入暫時附加元件」
3. 選擇 `manifest.json` 文件

## 相容平台

支援以下視頻平台：
- YouTube
- Vimeo
- Netflix
- Twitch
- 以及所有使用 HTML5 video 的網站
