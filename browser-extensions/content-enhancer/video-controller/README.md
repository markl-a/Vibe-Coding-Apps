# Video Controller Pro 🎬

強大的視頻播放控制擴展，提供播放速度調整、截圖、畫中畫和**AI 增強功能**！

## ✨ 功能特色

### 🎮 基礎播放控制
- **變速播放**：0.25x - 4x 速度調整，精準控制
- **精確跳轉**：向前/後跳轉 5秒、10秒、30秒
- **音量控制**：快速調整音量
- **畫中畫模式**：支援 PiP 模式，多任務處理
- **循環播放**：AB 段重複播放
- **播放記憶**：自動記住播放位置

### 🤖 AI 增強功能 (NEW!)
- **AI 視頻分析**：
  - 自動分析視頻信息（時長、解析度、進度）
  - 智能生成關鍵時刻標記
  - 顯示觀看統計和剩餘時間
  - 可點擊的時間戳快速跳轉

- **實時字幕識別**：
  - 使用 Web Speech API 進行語音識別
  - 自動生成實時字幕
  - 支持多語言識別
  - 美觀的字幕顯示面板

### 📷 進階功能
- **視頻截圖**：一鍵捕捉當前畫面，自動下載
- **右鍵菜單**：快速訪問常用功能
- **統計追蹤**：記錄觀看時間和截圖數量

### 鍵盤快捷鍵
- `Space` - 播放/暫停
- `←` / `→` - 後退/前進 5秒
- `Shift + ←/→` - 後退/前進 10秒
- `↑` / `↓` - 增加/減少音量
- `[` / `]` - 減慢/加快播放速度
- `F` - 全螢幕
- `P` - 畫中畫
- `S` - 截圖
- `0-9` - 跳轉到 0%-90%

## 安裝方式

### Chrome/Edge
1. 打開 `chrome://extensions/`
2. 啟用「開發人員模式」
3. 點擊「載入未封裝項目」
4. 選擇此目錄

### Firefox
1. 打開 `about:debugging#/runtime/this-firefox`
2. 點擊「載入暫時附加元件」
3. 選擇 `manifest.json` 文件
