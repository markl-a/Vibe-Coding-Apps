# Font Customizer Pro 🔤

一個功能強大的瀏覽器擴充功能，讓你可以自訂任何網站的字體樣式，提升閱讀體驗和可讀性。

## ✨ 功能特色

- **字體家族選擇** - 支援多種字體，包括中英文字體
- **字體大小調整** - 10px ~ 32px 靈活調整
- **字體粗細控制** - 從細到極粗多種選擇
- **行高調整** - 優化文字間距，提升閱讀舒適度
- **字母間距** - 精細控制字母間距
- **即時預覽** - 即時查看設定效果
- **網域記憶** - 為不同網站儲存不同設定
- **快捷鍵** - Ctrl+Shift+F 快速切換

## 🚀 安裝方式

### 開發模式安裝

1. 打開 Chrome 瀏覽器
2. 進入 `chrome://extensions/`
3. 開啟右上角的「開發人員模式」
4. 點擊「載入未封裝項目」
5. 選擇此專案的資料夾

### 使用方式

1. 點擊瀏覽器工具列的擴充功能圖標
2. 開啟「啟用字體自訂」開關
3. 調整各項字體設定
4. 即時預覽效果
5. 點擊「儲存此網站」記住當前網站的設定

## 🎯 使用情境

- **閱讀困難** - 調整字體大小和行高，讓閱讀更輕鬆
- **視力保護** - 選擇合適的字體和間距，減少眼睛疲勞
- **個人偏好** - 統一所有網站的字體風格
- **無障礙需求** - 為有特殊需求的使用者優化字體顯示

## ⌨️ 快捷鍵

- `Ctrl + Shift + F` - 快速開關字體自訂

## 🛠️ 技術細節

- **Manifest Version**: 3
- **權限**:
  - `activeTab` - 存取當前標籤頁
  - `storage` - 儲存設定
  - `scripting` - 注入腳本
- **Content Scripts** - 在所有網頁上運行
- **Storage API** - 同步設定到 Chrome 帳號

## 📝 設定說明

### 支援的字體

- Arial
- Times New Roman
- Courier New
- Georgia
- Verdana
- Trebuchet MS
- Comic Sans MS
- Impact
- 微軟正黑體
- 蘋方繁體

### 設定範圍

- **字體大小**: 10px - 32px
- **行高**: 1.0 - 3.0
- **字母間距**: -2px - 10px
- **字體粗細**: 細、標準、中等、半粗、粗體、極粗
