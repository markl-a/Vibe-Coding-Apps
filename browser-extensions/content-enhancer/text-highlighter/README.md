# Text Highlighter Pro

專業的文本高亮和標註工具，支援多種顏色、筆記功能和雲端同步。

## 功能特色

### 核心功能
- **多色高亮**：5種預設高亮顏色，支援自訂
- **文字筆記**：為高亮文本添加筆記和註解
- **智能選取**：自動識別段落和句子邊界
- **標籤管理**：使用標籤組織高亮內容
- **搜尋高亮**：快速搜尋所有高亮文本

### 進階功能
- **雲端同步**：跨裝置同步高亮和筆記
- **匯出功能**：匯出為 Markdown、PDF 或純文字
- **分享連結**：生成可分享的高亮連結
- **快捷鍵**：提升標註效率
- **側邊欄**：查看頁面所有高亮
- **統計資訊**：分析高亮習慣

## 安裝方式

### Chrome/Edge
1. 打開 `chrome://extensions/`
2. 啟用「開發人員模式」
3. 點擊「載入未封裝項目」
4. 選擇此目錄

### Firefox
1. 打開 `about:debugging#/runtime/this-firefox`
2. 點擊「載入暫時附加元件」
3. 選擇 `manifest.json` 文件

## 使用說明

### 基本使用
1. 選取要高亮的文本
2. 點擊出現的高亮按鈕
3. 選擇顏色
4. （可選）添加筆記

### 高亮顏色
- 🟡 黃色 - 重要內容
- 🟢 綠色 - 已理解
- 🔵 藍色 - 需要研究
- 🔴 紅色 - 關鍵重點
- 🟣 紫色 - 疑問

### 鍵盤快捷鍵
- `Alt+H` - 快速高亮（使用預設顏色）
- `Alt+1-5` - 使用特定顏色高亮
- `Alt+N` - 為選取文本添加筆記
- `Alt+D` - 刪除高亮
- `Alt+S` - 開啟側邊欄

### 管理高亮
- 點擊高亮文本查看/編輯筆記
- 右鍵點擊可更改顏色或刪除
- 使用側邊欄查看所有高亮

## 匯出格式

### Markdown
```markdown
# 頁面標題

## 高亮內容

### 黃色高亮
- 文本內容
  > 筆記內容
```

### 純文字
```
頁面: 標題
網址: URL

[黃色] 文本內容
筆記: 筆記內容
```

## 隱私說明

- 所有資料儲存在本地或您的 Chrome 同步空間
- 不會上傳到第三方伺服器
- 可隨時匯出或刪除所有資料

## 技術特點

- **Range API**：精確的文本選取和定位
- **Mutation Observer**：處理動態網頁
- **Storage API**：可靠的資料同步
- **高效能**：使用虛擬化渲染大量高亮
