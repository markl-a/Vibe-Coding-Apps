# Clipboard Manager Pro 📋

一個功能強大的剪貼簿管理工具，幫助你儲存、組織和快速存取剪貼簿歷史。

## ✨ 功能特色

- **剪貼簿歷史** - 自動儲存最多 100 個複製的項目
- **智能搜尋** - 快速搜尋歷史記錄
- **釘選功能** - 釘選重要的項目永久保存
- **一鍵複製** - 點擊任何項目即可複製
- **篩選功能** - 按全部、已釘選、最近項目篩選
- **快捷鍵** - Ctrl+Shift+V 快速開啟
- **右鍵選單** - 快速存取功能
- **統計資訊** - 顯示總數、釘選數、今日新增數

## 🚀 使用方式

### 基本操作

1. **自動儲存**
   - 複製任何文字，自動加入歷史記錄
   - 最多儲存 100 個項目（釘選項目不計）

2. **查看歷史**
   - 點擊擴充功能圖標
   - 或使用快捷鍵 `Ctrl+Shift+V` (Mac: `Cmd+Shift+V`)

3. **複製項目**
   - 點擊任何歷史項目即可複製
   - 或點擊 📄 複製按鈕

4. **釘選項目**
   - 點擊 📌 按鈕釘選重要項目
   - 釘選的項目不會被自動清除
   - 釘選項目顯示為紫色背景

5. **刪除項目**
   - 點擊 🗑️ 按鈕刪除單個項目
   - 點擊右上角「清空」清除所有未釘選項目

### 搜尋功能

1. 在搜尋框中輸入關鍵字
2. 即時顯示匹配的結果
3. 搜尋結果會高亮顯示關鍵字
4. 快捷鍵 `Ctrl+F` 快速聚焦搜尋框

### 篩選功能

- **全部** - 顯示所有項目
- **📌 已釘選** - 只顯示釘選的項目
- **⏰ 最近** - 只顯示今天新增的項目

## ⌨️ 快捷鍵

- `Ctrl+Shift+V` (Mac: `Cmd+Shift+V`) - 開啟剪貼簿管理器
- `Ctrl+F` - 聚焦搜尋框

## 🔒 隱私說明

- **本地儲存** - 所有資料儲存在本地瀏覽器
- **無雲端同步** - 資料不會上傳到任何伺服器
- **無追蹤** - 不收集任何使用資料

## ⚠️ 注意事項

### 儲存限制
- 最多儲存 100 個未釘選項目
- 釘選項目不受限制
- 超過限制時，最舊的項目會被移除

### 安全建議
- 不要儲存敏感資訊（密碼、信用卡號等）
- 定期清理不需要的項目
- 小心分享或展示剪貼簿內容

### 權限說明
- `clipboardRead` - 讀取剪貼簿內容
- `clipboardWrite` - 寫入剪貼簿
- `storage` - 儲存歷史記錄
- `contextMenus` - 右鍵選單功能

## 🛠️ 技術細節

### 自動監控
- 每 2 秒檢查一次剪貼簿
- 自動偵測新的複製內容
- 避免重複項目

### 資料結構
每個項目包含：
- 文字內容
- 類型標記
- 時間戳記
- 釘選狀態
- 唯一 ID

## 📝 常見問題

**Q: 為什麼有些內容沒有自動儲存？**
A: 擴充功能只能在有權限的情況下讀取剪貼簿。某些安全性較高的頁面可能限制訪問。

**Q: 釘選的項目會永久保存嗎？**
A: 是的，釘選的項目不會被自動清除，除非手動刪除。

**Q: 如何匯出剪貼簿歷史？**
A: 目前版本不支援匯出功能，未來版本會加入。

**Q: 可以同步到其他裝置嗎？**
A: 目前版本僅支援本地儲存，未來可能加入雲端同步功能。
