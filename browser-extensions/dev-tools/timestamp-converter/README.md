# ⏰ 時間戳轉換器 - Timestamp Converter

> Unix Timestamp 與日期時間的雙向轉換工具

一個功能完整的時間戳轉換工具，支援多種時間格式轉換、時間計算以及快速預設功能。

## ✨ 核心功能

### 1. 即時當前時間
- 🕐 即時顯示當前時間戳（秒和毫秒）
- 🌍 顯示本地時間和UTC時間
- 🔄 每秒自動更新

### 2. 時間戳轉日期
- **支援格式**：
  - 10位時間戳（秒）
  - 13位時間戳（毫秒）
- **輸出格式**：
  - 本地時間格式
  - UTC時間格式
  - ISO 8601標準格式
  - 相對時間（如：3天前）

### 3. 日期轉時間戳
- 📅 日期時間選擇器
- 🔢 輸出秒和毫秒格式
- 📋 ISO 8601格式

### 4. 時間計算器
- ➕ 增加/減少時間
- 🧮 支援單位：
  - 秒、分鐘、小時
  - 天、週
  - 月、年
- 📊 顯示計算結果和新時間戳

### 5. 快速預設
- 🕐 現在
- 🌅 今天開始（00:00:00）
- 🌆 今天結束（23:59:59）
- 📅 昨天
- 📆 明天
- 📋 一週前
- 📊 一個月前
- 📈 一年前

## 🎨 介面說明

### 主要區塊
1. **當前時間區** - 即時顯示當前各種格式的時間

### 操作按鈕
- **轉換** - 執行轉換操作
- **清除** - 清空輸入和結果
- **使用當前** - 使用當前時間戳
- **貼上** - 從剪貼簿貼上
- **📋** - 複製結果到剪貼簿

## 🔧 技術細節

### 時間戳格式
- **Unix時間戳（秒）**: 自1970-01-01 00:00:00 UTC以來的秒數
- **Unix時間戳（毫秒）**: 自1970-01-01 00:00:00 UTC以來的毫秒數

### 時區處理
- 自動使用瀏覽器的本地時區
- UTC時間統一標準參考
- 時區差異自動處理

### 相對時間計算
```javascript
相對時間邏輯:
< 60秒   -> X 秒前
< 60分   -> X 分鐘前
< 24小時 -> X 小時前
< 30天   -> X 天前
< 12月   -> X 個月前
>= 1年   -> X 年前
```

## 🐛 錯誤處理

### 常見錯誤提示
- ❌ "請輸入時間戳" - 輸入框為空
- ❌ "無效的時間戳" - 格式不正確
- ❌ "時間戳超出合理範圍" - 不在1970-2100範圍內
- ❌ "無法轉換為有效日期" - 時間戳無效
