# 🔍 正規表示式測試器

強大的正規表示式測試工具，支援即時測試、群組擷取和詳細說明。

## ✨ 功能特色

- ✅ **即時測試** - 輸入即時顯示匹配結果
- ✅ **旗標支援** - g, i, m, s, u 旗標
- ✅ **群組擷取** - 顯示所有擷取群組
- ✅ **模式說明** - 自動解釋正規表示式組成
- ✅ **快速參考** - 完整的正規表示式語法參考
- ✅ **常用模式** - 內建 8+ 種常用模式
- ✅ **測試案例** - 預設測試案例快速驗證

## 🚀 快速開始

在瀏覽器中開啟 `index.html` 即可使用。

## 📋 主要功能

### 1. 正規表示式輸入
- 支援完整的 JavaScript 正規表示式語法
- 旗標選項: g (全域), i (忽略大小寫), m (多行), s (dotAll), u (Unicode)
- 即時語法錯誤檢查

### 2. 測試字串
- 多行文字輸入
- 即時高亮顯示匹配結果
- 顯示匹配數量和位置

### 3. 匹配結果
- 顯示所有匹配項目
- 顯示匹配位置
- 顯示擷取群組內容

### 4. 快速參考
分類顯示正規表示式語法：
- **基本** - 常用字元類別
- **字元類別** - [abc], [^abc], [a-z]
- **數量詞** - *, +, ?, {n,m}
- **群組** - (abc), (?:abc), (a|b)
- **錨點** - ^, $, \\b, \\B

### 5. 常用模式
- 英文字母
- 數字
- 電話號碼
- Email
- URL
- 強密碼
- HEX 顏色
- 日期格式

## 💡 使用範例

### 範例 1: 驗證 Email
```
正規表示式: ^[\w-\.]+@([\w-]+\.)+[\w-]{2,4}$
測試字串: test@example.com ✓
測試字串: invalid@ ✗
```

### 範例 2: 提取電話號碼
```
正規表示式: [0-9]{3}-[0-9]{4}
旗標: g
測試字串: 請撥打 123-4567 或 987-6543
結果: 2 個匹配
```

### 範例 3: 群組擷取
```
正規表示式: (\d{4})-(\d{2})-(\d{2})
測試字串: 2024-01-15
結果:
  完整匹配: "2024-01-15"
  群組 1: "2024"
  群組 2: "01"
  群組 3: "15"
```

## 🎯 常用正規表示式

### Email
```regex
^[\w-\.]+@([\w-]+\.)+[\w-]{2,4}$
```

### URL
```regex
^https?:\/\/(www\.)?[-a-zA-Z0-9@:%._\+~#=]{1,256}\.[a-zA-Z0-9()]{1,6}\b
```

### 電話號碼（台灣）
```regex
[0-9]{2,4}-[0-9]{6,8}
```

### 日期 YYYY-MM-DD
```regex
^\d{4}-\d{2}-\d{2}$
```

### 強密碼（8+ 字元，含字母和數字）
```regex
^(?=.*[A-Za-z])(?=.*\d)[A-Za-z\d]{8,}$
```

## 📚 正規表示式語法快速參考

### 字元類別
- `.` - 任何字元
- `\d` - 數字 [0-9]
- `\w` - 文字字元 [a-zA-Z0-9_]
- `\s` - 空白字元

### 數量詞
- `*` - 0 次或多次
- `+` - 1 次或多次
- `?` - 0 次或 1 次
- `{n}` - 恰好 n 次
- `{n,m}` - n 到 m 次

### 錨點
- `^` - 字串開頭
- `$` - 字串結尾
- `\b` - 單字邊界
