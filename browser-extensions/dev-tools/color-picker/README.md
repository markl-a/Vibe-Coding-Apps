# 🎨 顏色選擇器

專業的顏色選擇、轉換和管理工具，支援多種顏色格式和色彩調和功能。

## ✨ 功能特色

- ✅ **顏色選擇** - 原生顏色選擇器 + RGB/HSL 滑桿
- ✅ **格式轉換** - HEX, RGB, RGBA, HSL, HSLA, CSS Filter
- ✅ **色彩調和** - 互補色、類似色、三角色、四角色、深淺色
- ✅ **對比度檢查** - WCAG 無障礙標準檢查
- ✅ **顏色歷史** - 自動記錄最近使用的顏色
- ✅ **預設顏色** - Material Design, Tailwind CSS, Web Safe 色彩集
- ✅ **一鍵複製** - 快速複製各種格式

## 🚀 快速開始

在瀏覽器中開啟 `index.html` 即可使用。

## 📋 主要功能

### 1. 顏色選擇
- 原生顏色選擇器
- RGB 滑桿調整 (Red, Green, Blue)
- HSL 滑桿調整 (Hue, Saturation, Lightness)
- Alpha 透明度調整

### 2. 格式轉換
支援多種顏色格式：
- **HEX**: #3B82F6
- **RGB**: rgb(59, 130, 246)
- **RGBA**: rgba(59, 130, 246, 1)
- **HSL**: hsl(217, 91%, 60%)
- **HSLA**: hsla(217, 91%, 60%, 1)
- **CSS Filter**: brightness(100%) contrast(100%)

### 3. 色彩調和
- **互補色** - 色輪上相對的顏色
- **類似色** - 相鄰的顏色
- **三角色** - 120° 間隔的三種顏色
- **四角色** - 90° 間隔的四種顏色
- **深淺色** - 同色系的不同明度

### 4. 對比度檢查器
- 即時計算前景色與背景色的對比度
- WCAG 無障礙標準檢查：
  - AA (大字): 3:1
  - AA (小字): 4.5:1
  - AAA (大字): 4.5:1
  - AAA (小字): 7:1

### 5. 顏色歷史
- 自動記錄最近使用的 20 種顏色
- 點擊歷史顏色快速選用
- 可清除歷史記錄

### 6. 預設色彩集
- **Material Design** - Google Material Design 色彩
- **Tailwind CSS** - Tailwind 預設色彩
- **Web Safe** - 網頁安全色彩

## 💡 使用技巧

### 技巧 1: 快速選色
使用原生顏色選擇器快速選擇基礎顏色，再用 RGB/HSL 滑桿微調。

### 技巧 2: 色彩調和
選擇主色後，使用色彩調和功能找到搭配的顏色，建立配色方案。

### 技巧 3: 無障礙設計
使用對比度檢查器確保文字與背景的對比度符合 WCAG 標準。

### 技巧 4: 預設顏色
從預設色彩集中選擇專業配色，快速應用於專案。
