# 📝 Markdown 預覽器 - Markdown Previewer

> 功能完整的 Markdown 即時預覽工具，支援 GitHub Flavored Markdown

一個強大的 Markdown 編輯器和預覽工具，提供即時渲染、語法高亮、多種檢視模式和導出功能。

## ✨ 核心功能

### 1. 即時預覽
- ⚡ 輸入即時渲染（300ms 防抖優化）
- 👁️ 同步滾動（計劃中）
- 🎨 美觀的預覽樣式
- 📱 響應式設計

### 2. 完整工具列
**標題按鈕**：
- H1, H2, H3 快速插入

**文字格式**：
- **粗體** (`**text**`)
- *斜體* (`*text*`)
- ~~刪除線~~ (`~~text~~`)
- `行內程式碼` (`` `code` ``)

**列表**：
- 無序列表
- 有序列表
- 待辦事項 (`- [ ]`)

**插入元素**：
- 🔗 連結
- 🖼️ 圖片
- ❝ 引用
- 📋 程式碼區塊
- ─ 水平分隔線
- 📊 表格

### 3. 語法高亮
- 🎨 使用 Highlight.js
- 💻 支援 190+ 程式語言
- 🎯 自動語言檢測
- 🌈 深色/淺色主題切換

### 4. 檢視模式
- **分割檢視** - 同時顯示編輯器和預覽
- **純編輯模式** - 全螢幕編輯
- **純預覽模式** - 全螢幕預覽
- 🔄 一鍵切換

### 5. 導出功能
- **下載 Markdown** (.md) - 保存原始 Markdown 文件
- **下載 HTML** (.html) - 完整的獨立 HTML 檔案（含樣式）
- **複製 HTML** - 複製渲染後的 HTML 到剪貼簿

### 6. 智能統計
- 字元數統計
- 字數統計
- 行數統計
- 即時更新

### 7. 快速參考
- 📚 內建 Markdown 語法參考
- 💡 常用語法範例
- 🎯 一鍵收合/展開

## 🚀 快速開始

### 直接使用
```bash
# 在瀏覽器中打開
open index.html
```

### 範例用法
1. 點擊「範例」按鈕載入示範文件
2. 或開始輸入你的 Markdown
3. 右側即時顯示預覽

## 💻 使用方式

### 基本操作

#### 插入格式
1. **使用工具列按鈕** - 點擊對應按鈕插入語法
2. **鍵盤輸入** - 直接輸入 Markdown 語法
3. **選取文字** - 選取文字後點擊格式按鈕

#### 快捷鍵
- `Tab` - 插入 4 個空格（縮排）

### 檢視模式切換
點擊右上角的 **⊞** 按鈕切換檢視模式：
- ⊞ 分割檢視
- ✏️ 純編輯模式
- 👁️ 純預覽模式

### 主題切換
點擊右上角的 **🌙**/**☀️** 按鈕切換深色/淺色主題

### 導出文件
在預覽面板的工具列：
- **📋 複製 HTML** - 複製渲染後的 HTML
- **⬇️ 下載 MD** - 下載 Markdown 文件
- **⬇️ 下載 HTML** - 下載完整的 HTML 文件

## 📖 Markdown 語法範例

### 標題
```markdown
# H1 標題
## H2 標題
### H3 標題
```

### 文字樣式
```markdown
**粗體文字**
*斜體文字*
***粗斜體***
~~刪除線~~
`行內程式碼`
```

### 列表
```markdown
無序列表：
- 項目 1
- 項目 2
  - 子項目 2.1

有序列表：
1. 第一項
2. 第二項

待辦事項：
- [x] 已完成
- [ ] 待完成
```

### 連結和圖片
```markdown
[連結文字](https://example.com)
![圖片說明](image-url.jpg)
```

### 引用
```markdown
> 這是一段引用文字
> 可以跨多行
```

### 程式碼
```markdown
行內程式碼：`const x = 10;`

程式碼區塊：
\`\`\`javascript
function hello() {
    console.log('Hello, World!');
}
\`\`\`
```

### 表格
```markdown
| 欄位1 | 欄位2 | 欄位3 |
|-------|-------|-------|
| 內容1 | 內容2 | 內容3 |
| 內容4 | 內容5 | 內容6 |
```

- ✅ Chrome 90+
- ✅ Firefox 88+
- ✅ Safari 14+
- ✅ Edge 90+

### 關鍵特性
```javascript
// Marked.js 設定
marked.setOptions({
    breaks: true,          // 支援 GFM 換行
    gfm: true,             // GitHub Flavored Markdown
    smartLists: true,      // 智能列表
    smartypants: true,     // 智能標點符號
    highlight: function    // 程式碼高亮
});
```
