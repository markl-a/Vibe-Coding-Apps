# 📱 Hashtag Generator Pro

> 🤖 **AI驅動的智能標籤生成器** - 為你的社交媒體貼文生成最相關的熱門標籤

## ✨ 功能特色

### 🎯 核心功能
- **多平台支持** - Instagram、Twitter、TikTok、YouTube
- **智能生成** - 基於主題和關鍵字生成相關標籤
- **熱門標籤** - 自動包含平台熱門標籤
- **可調數量** - 5-30個標籤任意選擇
- **一鍵複製** - 快速複製所有或選擇的標籤

### 🤖 AI 增強功能（新增！）
- **關鍵字提取** - 智能提取主題中的核心關鍵字
- **概念關聯** - 基於語義關聯推薦相關標籤
- **相關性排序** - 按相關度自動排序標籤
- **利基標籤** - 生成針對性的小眾標籤
- **參與度優化** - 包含提升互動的標籤

### 📊 實用功能
- **標籤統計** - 即時顯示標籤數量和字元數
- **選擇性複製** - 點擊選擇想要的標籤
- **生成歷史** - 保存最近50次生成記錄
- **趨勢標記** - 高亮顯示熱門標籤

## 🚀 使用方式

### 基本操作

1. **輸入主題**
   - 在輸入框中輸入關鍵字
   - 可以輸入多個關鍵字，用空格或逗號分隔
   - 例如：旅遊、美食、攝影

2. **選擇平台**
   - Instagram 📷
   - Twitter 🐦
   - TikTok 🎵
   - YouTube 📺

3. **調整設定**
   - 使用滑桿選擇標籤數量（5-30個）
   - 勾選「包含熱門標籤」加入平台熱門標籤

4. **生成標籤**
   - 點擊「生成標籤」按鈕
   - 查看生成的標籤列表

5. **選擇和複製**
   - 點擊標籤可切換選擇狀態
   - 點擊「複製已選擇的標籤」複製選定標籤
   - 點擊「複製全部」複製所有標籤

## 📊 平台建議

### Instagram 📷
- **建議數量**: 20-30 個標籤
- **策略**: 混合熱門標籤和小眾標籤
- **放置**: 貼文說明或第一則留言
- **效果**: 提升貼文在搜尋和探索頁面的曝光

### Twitter 🐦
- **建議數量**: 1-2 個標籤
- **策略**: 使用精準且相關的標籤
- **放置**: 融入推文內容
- **效果**: 參與趨勢話題討論

### TikTok 🎵
- **建議數量**: 4-5 個標籤
- **策略**: 使用 #fyp #foryou 加上相關標籤
- **放置**: 影片說明
- **效果**: 增加進入 For You 頁面的機會

### YouTube 📺
- **建議數量**: 15-20 個標籤
- **策略**: 包含頻道名稱和內容相關標籤
- **放置**: 影片說明和標籤欄位
- **效果**: 提升搜尋結果排名

## 💡 標籤策略

### 熱門標籤（高競爭）
- 例如：#love #instagood #photooftheday
- **優點**: 瀏覽量大
- **缺點**: 競爭激烈，內容快速被淹沒
- **建議**: 佔 30-40%

### 中等標籤（中競爭）
- 例如：#travelphotography #foodstagram
- **優點**: 平衡曝光和競爭
- **缺點**: 需要精準選擇
- **建議**: 佔 40-50%

### 小眾標籤（低競爭）
- 例如：#taipeitravel #taiwanfood
- **優點**: 目標受眾精準，停留時間長
- **缺點**: 瀏覽量相對較小
- **建議**: 佔 20-30%

## ⚠️ 注意事項

### 不要過度使用
- Instagram: 最多 30 個標籤
- Twitter: 避免超過 2 個
- TikTok: 保持在 5 個以內
- YouTube: 限制在 500 字元內

### 確保相關性
- 只使用與內容直接相關的標籤
- 避免使用誤導性標籤
- 不要濫用熱門標籤

### 注意格式
- 標籤不能有空格
- 只能使用字母、數字和底線
- 不要使用特殊符號
