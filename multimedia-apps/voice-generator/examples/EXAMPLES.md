# 🗣️ Voice Generator - 使用示例

## 快速开始示例

### 示例 1: 基础文字转语音

1. 启动应用：`npm start`
2. 在文本区域输入：
   ```
   欢迎使用 AI 语音生成器！
   ```
3. 选择音色：`Microsoft Yaoyao - Chinese (Simplified, PRC)`（中文女声）
4. 保持默认参数：
   - 语速：1.0x
   - 音调：1.0
   - 音量：100%
5. 点击 **预览** 按钮
6. 满意后点击 **导出音频**，选择 WAV 格式

**预期效果**：清晰的中文女声朗读

---

### 示例 2: 英文播客开场

1. 输入文本：
   ```
   Welcome to the Tech Talk Podcast!
   Today, we're discussing the latest trends in artificial intelligence.
   Stay tuned for an exciting episode!
   ```
2. 选择音色：`Microsoft David - English (United States)`（英文男声）
3. 调整参数：
   - 语速：0.9x（稍慢，更清晰）
   - 音调：0.9（低沉）
   - 音量：100%
4. 点击 **预览**
5. 导出为 MP3 格式

**预期效果**：专业的播客开场白，声音低沉有力

---

### 示例 3: 使用 SSML 增强表现力

1. 启用 SSML 模式（勾选 "启用 SSML"）
2. 输入 SSML 文本：
   ```xml
   <speak>
     <p>
       <s>这是一个<emphasis level="strong">非常重要</emphasis>的通知。</s>
       <break time="500ms"/>
       <s>请大家<prosody rate="slow">仔细聆听</prosody>。</s>
     </p>
   </speak>
   ```
3. 选择中文音色
4. 点击 **预览**

**预期效果**：带有强调和停顿的富有表现力的语音

---

## 导出格式选择

### WAV
- **质量**: 最高（无损）
- **文件大小**: 大
- **用途**: 专业制作、后期处理
- **推荐**: 需要编辑的项目

### MP3
- **质量**: 高（有损压缩）
- **文件大小**: 中等
- **用途**: 播客、网页、分享
- **推荐**: 一般用途

### OGG
- **质量**: 高
- **文件大小**: 小
- **用途**: 网页、游戏
- **推荐**: 开源项目

## 技巧和最佳实践

### ✅ 推荐做法
1. 使用标点符号控制停顿
2. 段落分明，便于理解
3. 先预览再导出
4. 保存常用设置
5. 使用 SSML 增强表现力

### ❌ 避免
1. 一次输入过长文本（建议 < 500 字）
2. 使用过快或过慢的语速
3. 极端的音调设置
4. 忽略标点符号

## 故障排除

**Q: 没有可用的音色？**
A: 确保系统已安装 TTS 引擎，Windows 10+ 自带。

**Q: 导出的音频没有声音？**
A: 检查音量设置，确保预览时能听到声音。

**Q: 中文发音不准确？**
A: 尝试其他中文音色，或使用汉语拼音辅助。

**Q: SSML 不起作用？**
A: 确保已勾选 "启用 SSML"，检查 XML 语法。
