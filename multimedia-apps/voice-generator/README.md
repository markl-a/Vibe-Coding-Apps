# Voice Generator - 语音生成器

专业的文字转语音（TTS）工具，支持多种音色、实时预览、参数调节和音频导出功能。

## 功能特性

### 核心功能
- **文字转语音（TTS）** - 将文本转换为自然流畅的语音
- **多种音色** - 支持男声、女声及多种语言的语音音色
- **实时预览** - 即时播放生成的语音效果
- **参数调节** - 精确控制语速、音调和音量
- **波形可视化** - 实时显示音频波形
- **音频导出** - 支持导出为 MP3、WAV 等格式
- **SSML 支持** - 使用 SSML 标记增强语音表现力

### 界面特性
- 现代化 UI 设计
- 直观的参数控制面板
- 实时波形显示
- 音色选择器
- 文本编辑器（支持 SSML）
- 播放控制

## 使用说明

### 基本使用

1. **输入文本**
   - 在文本区域输入要转换的文字
   - 支持纯文本和 SSML 标记

2. **选择音色**
   - 从音色列表中选择所需的语音
   - 支持不同性别和语言的音色

3. **调节参数**
   - **语速**：0.1x - 3.0x（默认 1.0x）
   - **音调**：0.1 - 2.0（默认 1.0）
   - **音量**：0.0 - 1.0（默认 1.0）

4. **预览和导出**
   - 点击"预览"按钮试听效果
   - 点击"导出音频"保存为文件

### SSML 标记示例

```xml
<speak>
  <p>欢迎使用语音生成器。</p>
  <break time="500ms"/>
  <p>这是一个<emphasis level="strong">强调</emphasis>的例子。</p>
  <p>我可以说得<prosody rate="slow">很慢</prosody>，</p>
  <p>也可以说得<prosody rate="fast">很快</prosody>。</p>
</speak>
```

### 支持的 SSML 标签

- `<speak>` - 根元素
- `<p>` - 段落
- `<s>` - 句子
- `<break>` - 停顿
- `<emphasis>` - 强调
- `<prosody>` - 韵律控制（rate, pitch, volume）
- `<say-as>` - 指定读法

## 快捷键

- `Ctrl/Cmd + P` - 预览语音
- `Ctrl/Cmd + S` - 导出音频
- `Ctrl/Cmd + L` - 清空文本
- `Space` - 播放/暂停（焦点在控制区域时）

## 音色列表

### 中文音色
- **小云（女声）** - 标准普通话，温柔亲切
- **小刚（男声）** - 标准普通话，沉稳大气
- **小梦（女声）** - 甜美可爱
- **晓晓（女声）** - 专业播音

### 英文音色
- **Emma（女声）** - 美式英语
- **Brian（男声）** - 英式英语
- **Joanna（女声）** - 美式英语，新闻风格
- **Matthew（男声）** - 美式英语

### 其他语言
- **日语** - さくら（女声）、けんじ（男声）
- **韩语** - 지은（女声）、민호（男声）
- **法语** - Céline（女声）、Mathieu（男声）

## 导出格式

支持导出以下音频格式：
- **MP3** - 通用格式，文件小
- **WAV** - 无损格式，质量高
- **OGG** - 开源格式

## 技术架构

### 核心技术
- **Electron** - 跨平台桌面应用框架
- **Web Speech API** - 浏览器原生语音合成
- **Canvas API** - 波形可视化
- **File System API** - 文件导出

## 已知问题

1. **音色可用性**
   - 可用的语音音色取决于操作系统
   - 某些高级音色可能需要下载语音包

2. **SSML 支持**
   - SSML 支持程度因浏览器而异
   - 建议使用 Chrome 或基于 Chromium 的浏览器

3. **导出限制**
   - 长文本可能需要较长处理时间
   - 文件大小取决于音频长度和格式

## 常见问题

**Q: 为什么没有听到声音？**
A: 检查系统音量设置，确保音频输出设备正常工作。

**Q: 如何添加更多音色？**
A: 音色由操作系统提供，可以在系统设置中下载额外的语音包。

**Q: 导出的音频文件在哪里？**
A: 默认保存在系统的下载文件夹，也可以在导出时选择位置。

**Q: 支持哪些语言？**
A: 支持中文、英文、日文、韩文、法文等多种语言，具体取决于系统安装的语音包。
