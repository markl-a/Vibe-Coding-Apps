# 🎵 Music Player - AI-Native 音樂播放器

基於 Electron 開發的跨平台音樂播放器,支援多種音頻格式、播放列表管理和均衡器功能。

## ✨ 功能特色

- 🎼 支援多種音頻格式 (MP3, WAV, FLAC, AAC, OGG, M4A)
- 📝 播放列表管理 (新增、刪除、排序)
- 🎚️ 10段均衡器 (多種預設模式)
- 🔀 播放模式 (順序、隨機、單曲循環)
- 📊 音頻可視化 (波形、頻譜)
- ⏱️ 進度控制與時間顯示
- 🎨 現代化 UI 設計
- ⌨️ 全面的鍵盤快捷鍵
- 💾 自動儲存播放列表
- 🎧 低延遲音頻處理

## 🛠️ 技術棧

- **框架**: Electron 28+
- **音頻處理**: Web Audio API
- **UI**: HTML5 + CSS3 + JavaScript
- **資料儲存**: LocalStorage / JSON
- **構建工具**: electron-builder

## ⌨️ 快捷鍵

| 快捷鍵 | 功能 |
|--------|------|
| `Space` | 播放 / 暫停 |
| `→` | 下一首 |
| `←` | 上一首 |
| `↑` | 增加音量 |
| `↓` | 減少音量 |
| `M` | 靜音 / 取消靜音 |
| `S` | 切換隨機播放 |
| `R` | 切換循環模式 |
| `Ctrl/Cmd + O` | 開啟音樂檔案 |
| `Ctrl/Cmd + L` | 清空播放列表 |
| `Ctrl/Cmd + E` | 開啟均衡器 |

## 🎯 核心功能實現

### 1. 音頻播放
- Web Audio API 提供低延遲播放
- 支援多種音頻格式
- 精確的時間控制

### 2. 播放列表
- 拖拽排序
- 批量導入
- 持久化儲存

### 3. 均衡器
- 10段圖形均衡器
- 預設音效模式 (古典、搖滾、爵士等)
- 自定義設定

### 4. 視覺效果
- 實時波形顯示
- 頻譜分析器
- Canvas 動畫

## 🎨 均衡器預設

- 🎸 **搖滾**: 增強低音和高音
- 🎹 **古典**: 強調中頻
- 🎺 **爵士**: 平衡的音色
- 🎧 **流行**: 適合流行音樂
- 🔊 **重低音**: 超強低音
- 🎼 **人聲**: 突出人聲頻段
