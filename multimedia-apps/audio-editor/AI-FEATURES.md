# 🤖 AI 增強功能 - Audio Editor

## 智能降噪

- **自適應降噪**: AI 自動檢測噪音類型並應用最佳降噪算法
- **人聲保護**: 在降噪時保護人聲不受影響
- **頻譜分析**: 可視化噪音頻譜並精確移除
- **預設模板**: 提供多種場景的降噪預設

## 智能音源分離

- **人聲分離**: 將人聲與伴奏分離
- **樂器識別**: 自動識別並分離不同樂器軌道
- **多軌導出**: 分離後可單獨導出各軌道
- **質量優化**: AI 優化分離質量

## 自動音量標準化

- **響度檢測**: 檢測音頻響度並標準化
- **峰值限制**: 自動應用峰值限制器防止削波
- **動態範圍壓縮**: 智能壓縮動態範圍
- **廣播標準**: 符合 LUFS 響度標準

## 智能剪輯建議

- **靜音檢測**: 自動檢測並標記靜音段落
- **呼吸聲移除**: 智能移除不必要的呼吸聲
- **填充詞檢測**: 識別「嗯」、「啊」等填充詞
- **時間優化**: 建議可以加速的段落

## 音效智能推薦

- **場景匹配**: 根據內容推薦合適的音效
- **混響設置**: AI 推薦最佳混響參數
- **均衡器調整**: 自動優化頻率響應
- **動態效果**: 智能應用壓縮、限幅等效果

## 轉錄與字幕

- **語音轉文字**: 將音頻內容轉錄為文字
- **時間戳生成**: 自動生成準確的時間戳
- **多語言支持**: 支持多種語言的轉錄
- **標點符號**: 智能添加標點符號

## 使用示例

```javascript
const ai = new AudioEditorAI();

// 智能降噪
const cleaned = await ai.denoise(audioBuffer, {
  aggressiveness: 'medium',
  preserveVocals: true
});

// 人聲分離
const separated = await ai.separateVocals(audioBuffer);
console.log('人聲軌道:', separated.vocals);
console.log('伴奏軌道:', separated.instrumental);

// 自動音量標準化
const normalized = await ai.normalizeLoudness(audioBuffer, {
  targetLUFS: -16,
  truePeak: -1
});
```

- **失真修復**: 修復過載和削波
- **頻率擴展**: 智能擴展頻率響圍
- **相位校正**: 自動校正相位問題
