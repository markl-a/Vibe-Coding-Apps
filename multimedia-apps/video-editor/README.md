# 🎥 Video Editor - AI-Native 視頻編輯器

功能完整的視頻編輯工具,支援剪輯、特效、字幕和轉場。

## ✨ 功能特色

- ✂️ 視頻剪輯 (剪切、分割、合併)
- 🎬 多軌時間軸
- 🎨 濾鏡與特效
- 🔄 轉場效果
- 📝 字幕編輯
- 🎵 音軌管理
- 🖼️ 圖片疊加
- ⚡ 實時預覽
- 📤 多格式匯出
- 🎯 關鍵幀動畫

## 🛠️ 技術棧

- **框架**: Electron 28+
- **視頻處理**: FFmpeg
- **預覽**: HTML5 Video
- **時間軸**: 自訂 Canvas 實現
- **UI**: HTML5 + CSS3 + JavaScript

## ⌨️ 快捷鍵

| 快捷鍵 | 功能 |
|--------|------|
| `Space` | 播放 / 暫停 |
| `I` | 設定入點 |
| `O` | 設定出點 |
| `C` | 剪切 |
| `Ctrl + Z` | 撤銷 |
| `Ctrl + Y` | 重做 |
| `Ctrl + S` | 儲存專案 |
| `Ctrl + E` | 匯出視頻 |

## 📁 專案結構

```
video-editor/
├── src/
│   ├── main.js          # Electron 主進程
│   ├── renderer.js      # 渲染進程邏輯
│   ├── timeline.js      # 時間軸編輯器
│   ├── videoEngine.js   # 視頻處理引擎
│   ├── effects.js       # 特效系統
│   ├── index.html
│   └── styles.css
├── package.json
└── README.md
```

## 🎯 核心功能

- **多軌編輯**: 視頻、音頻、字幕多軌道
- **即時預覽**: 編輯同時預覽效果
- **豐富特效**: 濾鏡、轉場、動畫
- **專業匯出**: 自訂編碼參數
