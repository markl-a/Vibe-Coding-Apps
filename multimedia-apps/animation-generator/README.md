# Animation Generator - 专业动画生成器

一个功能强大的桌面应用，用于创建各种类型的动画效果并导出为多种格式。

## 功能特性

### 动画类型
- **粒子效果** - 烟花、雨雪、星空、魔法等粒子系统
- **形状动画** - 几何形状的变换、旋转、缩放动画
- **路径动画** - 物体沿自定义路径运动
- **文字动画** - 打字机、渐现、弹跳等文字效果
- **波形动画** - 音频可视化、波浪效果
- **粒子文字** - 粒子组成的文字效果

### 参数控制
- **速度控制** - 调整动画播放速度和帧率
- **颜色设置** - 支持渐变色和随机色彩
- **大小调节** - 动态调整元素尺寸
- **数量设置** - 控制粒子/元素数量
- **持续时间** - 设置动画循环时长
- **背景选项** - 透明/纯色/渐变背景

### 导出功能
- **GIF 动画** - 导出为 GIF 格式（支持透明）
- **MP4 视频** - 高质量 H.264 视频
- **WebM 视频** - Web 优化格式
- **PNG 序列** - 导出帧序列图片
- **WebP 动画** - 现代化图片格式

### 预设模板
- 烟花爆炸效果
- 下雨/下雪场景
- 星空闪烁
- 粒子爆炸
- 波浪涌动
- DNA 螺旋
- 几何变换
- 文字打字机

## 安装说明

### 前置要求
- Node.js 16.0 或更高版本
- npm 或 yarn 包管理器

### 安装步骤

1. 克隆或下载项目：
```bash
cd animation-generator
```

2. 安装依赖：
```bash
npm install
```

3. 启动应用：
```bash
npm start
```

## 使用指南

### 基本操作

1. **选择动画类型**
   - 从左侧面板选择想要的动画类型
   - 每种类型都有独特的参数设置

2. **调整参数**
   - 使用滑块和输入框调整各项参数
   - 实时预览会立即反映更改

3. **预览动画**
   - 点击"播放"按钮开始预览
   - 使用"暂停"和"重置"控制播放

4. **应用预设**
   - 选择预设模板快速开始
   - 可在预设基础上进一步调整

5. **导出动画**
   - 选择导出格式（GIF/MP4/WebM/PNG序列）
   - 设置分辨率和质量参数
   - 点击"导出"保存文件

### 快捷键

- `Space` - 播放/暂停
- `R` - 重置动画
- `Ctrl/Cmd + E` - 导出动画
- `Ctrl/Cmd + S` - 保存预设
- `Ctrl/Cmd + O` - 加载预设
- `F11` - 全屏预览

## 技术架构

### 技术栈
- **Electron** - 跨平台桌面应用框架
- **Canvas API** - 高性能 2D 图形渲染
- **Web Workers** - 后台处理和编码
- **MediaRecorder API** - 视频录制
- **gif.js** - GIF 编码（通过 Canvas）

### 核心模块

#### main.js
- Electron 主进程
- 窗口管理
- 文件系统操作
- 菜单和快捷键

#### renderer.js
- 渲染进程主逻辑
- UI 交互处理
- Canvas 渲染控制
- 导出功能实现

#### animationEngine.js
- 动画生成引擎
- 粒子系统
- 形状动画
- 路径计算
- 帧管理

#### styles.css
- 现代化 UI 设计
- 响应式布局
- 动画和过渡效果

## 性能优化

- **Canvas 离屏渲染** - 提高绘制性能
- **对象池** - 减少内存分配
- **requestAnimationFrame** - 流畅动画
- **Web Workers** - 导出时不阻塞 UI
- **帧率控制** - 可调节 FPS

## 导出格式详解

### GIF 动画
- 适用于简单动画和透明背景
- 支持 256 色
- 文件较小，兼容性好
- 可设置循环次数

### MP4 视频
- 高质量视频输出
- 支持 H.264 编码
- 适合社交媒体分享
- 文件大小适中

### WebM 视频
- Web 优化格式
- VP8/VP9 编码
- 支持透明通道
- 浏览器原生支持

### PNG 序列
- 无损质量
- 便于后期编辑
- 每帧独立文件
- 支持透明背景

## 常见问题

### Q: 导出的 GIF 文件很大怎么办？
A: 可以降低分辨率、减少帧率或减少颜色数量。

### Q: 支持导入自定义图片吗？
A: 当前版本专注于程序生成动画，后续版本会添加图片导入功能。

### Q: 可以保存自定义预设吗？
A: 可以，使用 Ctrl/Cmd + S 保存当前设置为预设。

### Q: 最大支持多大的分辨率？
A: 建议不超过 1920x1080，更大的分辨率可能影响性能。

### Q: 动画预览卡顿怎么办？
A: 降低粒子数量或复杂度，关闭不必要的效果。
