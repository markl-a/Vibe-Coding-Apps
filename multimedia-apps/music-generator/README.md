# 🎵 Music Generator - AI音樂生成器

使用算法和AI技術生成音樂的應用程式。

## ✨ 功能特色

### 生成模式

#### 1. 🎼 旋律生成
- **隨機旋律**: 基於音階和音樂理論生成悅耳旋律
- **馬可夫鏈**: 使用馬可夫鏈算法生成連貫旋律
- **遺傳算法**: 進化式生成獨特旋律模式

#### 2. 🥁 節奏生成
- **鼓點模式**: 自動生成鼓點節奏
- **節奏變化**: 智能節奏變換
- **複雜節拍**: 支持多種拍號

#### 3. 🎹 和弦進行
- **經典進行**: I-V-vi-IV 等常見和弦進行
- **爵士和弦**: 複雜的爵士和弦序列
- **自動和聲**: 根據旋律自動生成和弦

#### 4. 🎨 氛圍生成
- **環境音樂**: 冥想、放鬆音樂
- **電子音樂**: EDM、Techno 風格
- **古典風格**: 巴洛克、古典時期風格

## 🎛️ 參數控制

- **音階選擇**: 大調、小調、五聲音階、布魯斯音階等
- **速度(BPM)**: 60-200 BPM
- **音符長度**: 1/1, 1/2, 1/4, 1/8, 1/16
- **音色選擇**: 鋼琴、合成器、弦樂、打擊樂等
- **音樂長度**: 自定義小節數
- **複雜度**: 簡單、中等、複雜

## 🎼 音樂理論支持

- ✅ 多種音階和調式
- ✅ 和弦理論
- ✅ 音程關係
- ✅ 節奏模式
- ✅ 動態表情

## 💾 導出功能

- **WAV格式**: 無損音質
- **MP3格式**: 壓縮音頻
- **MIDI格式**: 音樂數據
- **項目保存**: 保存生成參數

## 🎹 實時預覽

- 即時播放生成的音樂
- 可視化頻譜顯示
- 音符序列顯示
- 波形顯示

## 🛠️ 技術棧

- **框架**: Electron 28+
- **音頻引擎**: Tone.js
- **音頻處理**: Web Audio API
- **UI**: HTML5 + CSS3 + JavaScript

## 🚀 快速開始

```bash
# 安裝依賴
npm install

# 啟動應用
npm start

# 開發模式
npm run dev
```
