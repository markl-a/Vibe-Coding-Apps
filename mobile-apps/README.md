# 移動應用 Mobile Applications

這個資料夾包含各種移動應用專案。

## 子資料夾說明

### react-native
- 跨平台移動應用 (iOS & Android)
- 使用 React Native 框架
- Expo 專案

### flutter
- 跨平台移動應用
- 使用 Flutter/Dart
- 高性能 UI

### ios-native
- iOS 原生應用
- Swift/SwiftUI
- Objective-C

### android-native
- Android 原生應用
- Kotlin/Jetpack Compose
- Java

## 適合開發的應用類型
- 社交應用
- 工具類 App
- 健康健身追蹤
- 電子商務應用
- 遊戲
- 音樂/影片播放器
- 新聞閱讀器
- 天氣應用
- 相機/照片編輯

## 技術棧
- React Native + Expo
- Flutter + Firebase
- SwiftUI + CloudKit
- Jetpack Compose + Room
