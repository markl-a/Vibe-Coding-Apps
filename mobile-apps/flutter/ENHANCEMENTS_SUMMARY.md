#### 推薦算法權重
```
總分 = 基礎評分(10) +
       類別匹配(20) +
       時間適配(15) +
       難度適配(10) +
       飲食符合(25) +
       過敏檢查(-50) +
       收藏相似度(5) +
       新鮮度(10) +
       隨機因子(5)
```

#### 健康評分算法
```
基礎分數：60分
+ 高蛋白（≥20g）：+10分
+ 高纖維（≥5g）：+10分
+ 適中卡路里（300-600）：+10分
+ 低脂肪（<10g）：+10分
- 高卡路里（>800）：-10分
- 高脂肪（>30g）：-10分
最終：0-100（鉗位處理）
```
