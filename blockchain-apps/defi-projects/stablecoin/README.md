# 💵 Stablecoin - 算法穩定幣協議

## 📋 專案簡介

基於超額抵押的穩定幣發行協議,類似 MakerDAO 的 DAI。

## ✨ 核心功能

### 1. 鑄造穩定幣
- 存入抵押品 (ETH, WBTC 等)
- 鑄造穩定幣
- 最低抵押率要求 (例如 150%)

### 2. 贖回抵押品
- 燒毀穩定幣
- 取回抵押品

### 3. 清算機制
- 監控抵押率
- 自動清算低於閾值的金庫
- 清算獎勵

### 4. 穩定機制
- 利率調整
- 清算拍賣
- 緊急關停

## 🛠️ 技術架構

```
contracts/
├── Stablecoin.sol           # 穩定幣 ERC20
├── VaultManager.sol         # 金庫管理
├── PriceOracle.sol          # 價格預言機
└── LiquidationEngine.sol    # 清算引擎
```

## 📝 核心概念

### 抵押債務頭寸 (CDP)
用戶存入抵押品並鑄造穩定幣的位置。

### 抵押率
抵押品價值 / 債務價值的比率。

### 穩定費用
類似利率,用於維持穩定幣錨定。

## 🔐 風險參數

- **最低抵押率**: 150%
- **清算比率**: 130%
- **清算罰金**: 13%
- **穩定費用**: 可調整 (例如 2% 年化)

## 🤖 AI 輔助功能

### 價格監控
- 實時監控抵押品價格
- 預警即將清算的金庫

### 參數優化
- 基於歷史數據優化風險參數
- 模擬不同市場條件
