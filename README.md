# Vibe-coding-Apps
使用AI輔助開發寫出來的App,網頁,桌面應用甚至是遊戲

## 📁 專案結構

這個儲存庫包含多種類型的應用程式專案，全部使用 AI 輔助開發工具建立。

### 🌐 [web-apps](./web-apps) - 網頁應用
包含各種網頁應用，包括：
- 作品集/部落格
- 電子商務平台
- 社交媒體應用
- 儀表板和分析工具
- 生產力工具
- 著陸頁

### 📱 [mobile-apps](./mobile-apps) - 移動應用
跨平台和原生移動應用：
- React Native 應用
- Flutter 應用
- iOS 原生應用
- Android 原生應用

### 💻 [desktop-apps](./desktop-apps) - 桌面應用
桌面應用程式：
- Electron 應用
- Tauri 應用
- 原生桌面應用

### 🎮 [games](./games) - 遊戲
各種平台的遊戲：
- 網頁遊戲
- 移動遊戲
- 桌面遊戲
- 遊戲引擎專案

### 🛠️ [tools-utilities](./tools-utilities) - 工具與實用程式
開發者工具和實用程式：
- CLI 工具
- 自動化腳本
- 資料處理工具
- 開發工具

### 🔌 [apis-backend](./apis-backend) - API 與後端
後端服務和 API：
- REST API
- GraphQL
- 微服務
- 無伺服器函數

### 🧩 [browser-extensions](./browser-extensions) - 瀏覽器擴充功能
瀏覽器擴充功能：
- Chrome 擴充功能
- Firefox 擴充功能
- 跨瀏覽器擴充功能

### 🤖 [ai-ml-projects](./ai-ml-projects) - AI/機器學習專案
AI 和機器學習應用：
- 聊天機器人
- 圖像處理
- 資料分析
- 自然語言處理

## 🚀 開始使用

1. 選擇你想要開發的應用類型
2. 進入相應的資料夾
3. 閱讀該資料夾的 README.md 了解詳細資訊
4. 開始使用 AI 輔助工具建立你的專案！
