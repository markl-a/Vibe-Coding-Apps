# API 與後端 APIs & Backend

這個資料夾包含各種 API 和後端服務專案。

## 子資料夾說明

### rest-api
- RESTful API 服務
- CRUD 操作
- 資源管理

### graphql
- GraphQL API
- 型別安全的查詢
- 單一端點

### microservices (微服務)
- 分散式服務架構
- 獨立部署的服務
- 服務間通訊

### serverless (無伺服器)
- AWS Lambda, Google Cloud Functions
- Vercel Functions
- Netlify Functions
- 事件驅動架構

## 適合開發的後端服務
- 使用者認證系統
- 內容管理系統 (CMS)
- 電子商務後端
- 即時通訊後端
- 檔案儲存服務
- 支付處理系統
- 通知服務
- 搜尋引擎
- 分析追蹤服務

## 技術棧
- Node.js (Express, NestJS, Fastify)
- Python (Django, FastAPI, Flask)

- Rust (Actix, Rocket)
